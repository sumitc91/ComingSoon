819. Most Common Word

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a string paragraph and a string array of the banned words banned, return the most frequent word that is not banned. It is guaranteed there is at least one word that is not banned, and that the answer is unique.

The words in paragraph are case-insensitive and the answer should be returned in lowercase.

Examples:

Example 1:

Input: paragraph = "Bob hit a ball, the hit BALL flew far after it was hit.", banned = ["hit"]
Output: "ball"
Explanation: 
"hit" occurs 3 times, but it is a banned word.
"ball" occurs twice (and no other word does), so it is the most frequent non-banned word in the paragraph. 
Note that words in the paragraph are not case sensitive,
that punctuation is ignored (even if adjacent to words, such as "ball,"), 
and that "hit" isn't the answer even though it occurs more because it is banned.

Example 2:

Input: paragraph = "a.", banned = []
Output: "a"

Constraints:

  • 1 ≤ paragraph.length ≤ 1000
  • paragraph consists of English letters, space ' ', or punctuation marks '!?',;.'
  • 0 ≤ banned.length ≤ 100
  • 1 ≤ banned[i].length ≤ 10
  • banned[i] consists of only lowercase English letters

Python Solution

class Solution:
    def mostCommonWord(self, paragraph: str, banned: List[str]) -> str:
        # Convert to lowercase and replace punctuation with spaces
        for c in "!?',;.":
            paragraph = paragraph.replace(c, " ")
        paragraph = paragraph.lower()
        
        # Create word frequency counter excluding banned words
        banned_set = set(banned)
        word_count = Counter()
        
        for word in paragraph.split():
            if word not in banned_set:
                word_count[word] += 1
                
        # Return most common word
        return max(word_count.items(), key=lambda x: x[1])[0]

Implementation Notes:

  • Uses Counter class for word frequency counting
  • Handles punctuation by replacing with spaces
  • Time complexity: O(n) where n is length of paragraph
  • Space complexity: O(m) where m is number of unique words

Java Solution

class Solution {
    public String mostCommonWord(String paragraph, String[] banned) {
        // Convert to lowercase and replace punctuation
        paragraph = paragraph.toLowerCase();
        paragraph = paragraph.replaceAll("[!?',;.]", " ");
        
        // Create set of banned words
        Set bannedSet = new HashSet<>(Arrays.asList(banned));
        
        // Count word frequencies
        Map wordCount = new HashMap<>();
        String[] words = paragraph.split("\\s+");
        
        for (String word : words) {
            if (!word.isEmpty() && !bannedSet.contains(word)) {
                wordCount.put(word, wordCount.getOrDefault(word, 0) + 1);
            }
        }
        
        // Find most frequent word
        String result = "";
        int maxCount = 0;
        
        for (Map.Entry entry : wordCount.entrySet()) {
            if (entry.getValue() > maxCount) {
                maxCount = entry.getValue();
                result = entry.getKey();
            }
        }
        
        return result;
    }
}

        for (char& c : paragraph) {
            if (c == '!' || c == '?' || c == ',' || c == ';' || c == '.' || c == '\'') {
                c = ' ';
            }

        stringstream ss(paragraph);

            if (!bannedSet.count(word)) {
                wordCount[word]++;
            }
        }
        
        // Find most frequent word
        string result;
        int maxCount = 0;
        
        for (const auto& pair : wordCount) {
            if (pair.second > maxCount) {
                maxCount = pair.second;
                result = pair.first;
            }
        }
        
        return result;
    }
};

JavaScript Solution

/**
 * @param {string} paragraph
 * @param {string[]} banned
 * @return {string}
 */
var mostCommonWord = function(paragraph, banned) {
    // Convert to lowercase and replace punctuation
    paragraph = paragraph.toLowerCase();
    paragraph = paragraph.replace(/[!?',;.]/g, ' ');
    
    // Create set of banned words
    const bannedSet = new Set(banned);
    
    // Count word frequencies
    const wordCount = new Map();
    const words = paragraph.split(/\s+/);
    
    for (let word of words) {
        if (word && !bannedSet.has(word)) {
            wordCount.set(word, (wordCount.get(word) || 0) + 1);
        }
    }
    
    // Find most frequent word
    let result = '';
    let maxCount = 0;
    
    for (let [word, count] of wordCount) {
        if (count > maxCount) {
            maxCount = count;
            result = word;
        }
    }
    
    return result;
};

C# Solution

public class Solution {
    public string MostCommonWord(string paragraph, string[] banned) {
        // Convert to lowercase and replace punctuation
        paragraph = paragraph.ToLower();
        foreach (char c in new[] {'!', '?', '\'', ',', ';', '.'}) {
            paragraph = paragraph.Replace(c, ' ');
        }
        
        // Create set of banned words
        var bannedSet = new HashSet(banned);
        
        // Count word frequencies
        var wordCount = new Dictionary();
        var words = paragraph.Split(new[] {' '}, StringSplitOptions.RemoveEmptyEntries);
        
        foreach (var word in words) {
            if (!bannedSet.Contains(word)) {
                wordCount[word] = wordCount.GetValueOrDefault(word) + 1;
            }
        }
        
        // Find most frequent word
        return wordCount.OrderByDescending(x => x.Value).First().Key;
    }
}

Implementation Notes:

  • Uses Dictionary for word frequency counting
  • Handles punctuation using string Replace
  • Time complexity: O(n)
  • Space complexity: O(m)
