50. Pow(x, n)

Medium

Problem Description

Implement pow(x, n), which calculates x raised to the power n (i.e., x^n).

Examples

Example 1:
Input: x = 2.00000, n = 10
Output: 1024.00000

Example 2:
Input: x = 2.10000, n = 3
Output: 9.26100

Example 3:
Input: x = 2.00000, n = -2
Output: 0.25000
Explanation: 2^-2 = 1/2^2 = 1/4 = 0.25

Python Solution


def myPow(x: float, n: int) -> float:
    def fastPow(x: float, n: int) -> float:
        if n == 0:
            return 1.0
        
        half = fastPow(x, n // 2)
        if n % 2 == 0:
            return half * half
        else:
            return half * half * x
    
    if n < 0:
        x = 1 / x
        n = -n
    return fastPow(x, n)

Java Solution


class Solution {
    public double myPow(double x, int n) {
        if (n == Integer.MIN_VALUE) {
            x = x * x;
            n = n/2;
        }
        
        if (n < 0) {
            x = 1/x;
            n = -n;
        }
        
        return fastPow(x, n);
    }
    
    private double fastPow(double x, int n) {
        if (n == 0) {
            return 1.0;
        }
        
        double half = fastPow(x, n/2);
        if (n % 2 == 0) {
            return half * half;
        } else {
            return half * half * x;
        }
    }
}

JavaScript Solution


/**
 * @param {number} x
 * @param {number} n
 * @return {number}
 */
var myPow = function(x, n) {
    const fastPow = (x, n) => {
        if (n === 0) {
            return 1.0;
        }
        
        const half = fastPow(x, Math.floor(n/2));
        if (n % 2 === 0) {
            return half * half;
        } else {
            return half * half * x;
        }
    };
    
    if (n < 0) {
        x = 1/x;
        n = -n;
    }
    
    return fastPow(x, n);
};

Complexity Analysis

  • Time Complexity: O(log n) using fast power algorithm
  • Space Complexity: O(log n) for recursion stack

Solution Explanation

This solution uses the fast power algorithm to efficiently calculate power. Here's how it works:

  • Handle edge cases:
    • Negative exponents: convert to positive
    • Integer minimum value: special handling
  • Fast power algorithm:
    • Base case: n = 0, return 1
    • Recursively calculate half power
    • Even power: multiply half by itself
    • Odd power: multiply by x once more

Key points:

  • Uses divide and conquer approach
  • Handles negative exponents
  • Avoids overflow with large exponents
  • Maintains precision with floating point
