869. Reordered Power of 2

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an integer n. We reorder the digits in any order (including the original order) such that the leading digit is not zero.

Return true if and only if we can do this so that the resulting number is a power of 2.

Examples:

Example 1:

Input: n = 1
Output: true

Example 2:

Input: n = 10
Output: false

Example 3:

Input: n = 16
Output: true
Explanation: 16 is 2⁴, and we can reorder it to get 16 which is still 2⁴.

Constraints:

  • 1 ≤ n ≤ 10⁹

Python Solution

class Solution:
    def reorderedPowerOf2(self, n: int) -> bool:
        # Count digits in n
        count_n = Counter(str(n))
        
        # Check all powers of 2 with same number of digits
        n_length = len(str(n))
        for i in range(31):  # 2^31 - 1 is max int
            power = 1 << i
            if len(str(power)) == n_length:
                if Counter(str(power)) == count_n:
                    return True
            elif len(str(power)) > n_length:
                break
        
        return False

Implementation Notes:

  • Uses Counter to compare digit frequencies
  • Time complexity: O(log² n)
  • Space complexity: O(log n)

Java Solution

class Solution {
    public boolean reorderedPowerOf2(int n) {
        // Count digits in n
        int[] count = countDigits(n);
        
        // Check all powers of 2
        for (int i = 0; i < 31; i++) {
            if (Arrays.equals(count, countDigits(1 << i))) {
                return true;
            }
        }
        
        return false;
    }
    
    private int[] countDigits(int n) {
        int[] count = new int[10];
        while (n > 0) {
            count[n % 10]++;
            n /= 10;
        }
        return count;
    }
}

JavaScript Solution

/**
 * @param {number} n
 * @return {boolean}
 */
var reorderedPowerOf2 = function(n) {
    // Count digits in n
    const countDigits = (num) => {
        const count = new Array(10).fill(0);
        while (num > 0) {
            count[num % 10]++;
            num = Math.floor(num / 10);
        }
        return count;
    };
    
    const count = countDigits(n);
    
    // Check all powers of 2
    for (let i = 0; i < 31; i++) {
        if (count.toString() === countDigits(1 << i).toString()) {
            return true;
        }
    }
    
    return false;
};

C# Solution

public class Solution {
    public bool ReorderedPowerOf2(int n) {
        // Count digits in n
        int[] count = CountDigits(n);
        
        // Check all powers of 2
        for (int i = 0; i < 31; i++) {
            if (Enumerable.SequenceEqual(count, CountDigits(1 << i))) {
                return true;
            }
        }
        
        return false;
    }
    
    private int[] CountDigits(int n) {
        int[] count = new int[10];
        while (n > 0) {
            count[n % 10]++;
            n /= 10;
        }
        return count;
    }
}

Implementation Notes:

  • Uses array to count digit frequencies
  • Time complexity: O(log² n)
  • Space complexity: O(1) as we use fixed-size arrays
