662. Maximum Width of Binary Tree

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given the root of a binary tree, return the maximum width of the given tree.

The maximum width of a tree is the maximum width among all levels.

The width of one level is defined as the length between the end-nodes (the leftmost and rightmost non-null nodes), where the null nodes between the end-nodes that would be present in a complete binary tree extending down to that level are also counted into the length calculation.

It is guaranteed that the answer will in the range of a 32-bit signed integer.

Examples:

Example 1:

Input: root = [1,3,2,5,3,null,9]
Output: 4
Explanation: The maximum width exists in the third level with length 4 (5,3,null,9).

Example 2:

Input: root = [1,3,2,5,null,null,9,6,null,7]
Output: 7
Explanation: The maximum width exists in the fourth level with length 7 (6,null,null,null,null,null,7).

Example 3:

Input: root = [1,3,2,5]
Output: 2
Explanation: The maximum width exists in the second level with length 2 (3,2).

Constraints:

  • The number of nodes in the tree is in the range [1, 3000]
  • -100 ≤ Node.val ≤ 100

Python Solution

class Solution:
    def widthOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        
        # Use level order traversal with position indexing
        max_width = 0
        queue = deque([(root, 0)])  # (node, position)
        
        while queue:
            level_size = len(queue)
            _, level_start = queue[0]
            
            # Process all nodes at current level
            for i in range(level_size):
                node, position = queue.popleft()
                
                # Add children with their positions
                if node.left:
                    queue.append((node.left, position * 2))
                if node.right:
                    queue.append((node.right, position * 2 + 1))
                
                # Update max width if this is last node in level
                if i == level_size - 1:
                    max_width = max(max_width, position - level_start + 1)
        
        return max_width

Alternative Solution (Using DFS):

class Solution:
    def widthOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        # Dictionary to store leftmost position at each level
        first_pos = {}
        self.max_width = 0
        
        def dfs(node: TreeNode, level: int, pos: int) -> None:
            if not node:
                return
            
            # Record leftmost position at this level
            if level not in first_pos:
                first_pos[level] = pos
            
            # Update max width using current position and leftmost position
            self.max_width = max(self.max_width, pos - first_pos[level] + 1)
            
            # Traverse children with updated positions
            dfs(node.left, level + 1, pos * 2)
            dfs(node.right, level + 1, pos * 2 + 1)
        
        dfs(root, 0, 0)
        return self.max_width

Implementation Notes:

  • First solution uses BFS with O(n) time and space
  • Second solution uses DFS with O(n) time and O(h) space
  • Both solutions handle integer overflow by tracking relative positions

Java Solution

class Solution {
    public int widthOfBinaryTree(TreeNode root) {
        if (root == null) {
            return 0;
        }
        
        int maxWidth = 0;
        Queue> queue = new LinkedList<>();
        queue.offer(new Pair<>(root, 0));
        
        while (!queue.isEmpty()) {
            int levelSize = queue.size();
            int levelStart = queue.peek().getValue();
            
            for (int i = 0; i < levelSize; i++) {
                Pair pair = queue.poll();
                TreeNode node = pair.getKey();
                int position = pair.getValue();
                
                if (node.left != null) {
                    queue.offer(new Pair<>(node.left, position * 2L));
                }
                if (node.right != null) {
                    queue.offer(new Pair<>(node.right, position * 2L + 1));
                }
                
                if (i == levelSize - 1) {
                    maxWidth = Math.max(maxWidth, position - levelStart + 1);
                }
            }
        }
        
        return maxWidth;
    }
}

JavaScript Solution

/**
 * @param {TreeNode} root
 * @return {number}

var widthOfBinaryTree = function(root) {
    if (!root) {
        return 0;
    }
    
    let maxWidth = 0;
    const queue = [[root, 0n]];  // Use BigInt for position
    
    while (queue.length) {
        const levelSize = queue.length;
        const levelStart = queue[0][1];
        
        for (let i = 0; i < levelSize; i++) {
            const [node, position] = queue.shift();
            
            if (node.left) {
                queue.push([node.left, position * 2n]);
            }
            if (node.right) {
                queue.push([node.right, position * 2n + 1n]);
            }
            
            if (i === levelSize - 1) {
                maxWidth = Math.max(maxWidth, Number(position - levelStart + 1n));
            }
        }
    }
    
    return maxWidth;
};

C# Solution

public class Solution {
    public int WidthOfBinaryTree(TreeNode root) {
        if (root == null) {
            return 0;
        }
        
        int maxWidth = 0;
        Queue<(TreeNode node, long position)> queue = new Queue<(TreeNode, long)>();
        queue.Enqueue((root, 0));
        
        while (queue.Count > 0) {
            int levelSize = queue.Count;
            long levelStart = queue.Peek().position;
            
            for (int i = 0; i < levelSize; i++) {
                var (node, position) = queue.Dequeue();
                
                if (node.left != null) {
                    queue.Enqueue((node.left, position * 2));
                }
                if (node.right != null) {
                    queue.Enqueue((node.right, position * 2 + 1));
                }
                
                if (i == levelSize - 1) {
                    maxWidth = Math.Max(maxWidth, (int)(position - levelStart + 1));
                }
            }
        }
        
        return maxWidth;
    }
}

Implementation Notes:

  • Uses level order traversal with position tracking
  • Handles large numbers using long/BigInt to prevent overflow
  • Time complexity: O(n), Space complexity: O(w) where w is max width
