661. Image Smoother

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

An image smoother is a filter of the size 3 x 3 that can be applied to each cell of an image by rounding down the average of the cell and the eight surrounding cells (i.e., the average of the nine cells in the blue smoother). If one or more of the surrounding cells of a cell is not present, we do not consider it in the average (i.e., the average of the four cells in the red smoother).

Given an m x n integer matrix img representing the grayscale of an image, return the image after applying the smoother on each cell of it.

Examples:

Example 1:

Input: img = [[1,1,1],[1,0,1],[1,1,1]]
Output: [[0,0,0],[0,0,0],[0,0,0]]
Explanation:
For the points (0,0), (0,2), (2,0), (2,2): floor(3/4) = floor(0.75) = 0
For the points (0,1), (1,0), (1,2), (2,1): floor(5/6) = floor(0.83333333) = 0
For the point (1,1): floor(8/9) = floor(0.88888889) = 0

Example 2:

Input: img = [[100,200,100],[200,50,200],[100,200,100]]
Output: [[137,141,137],[141,138,141],[137,141,137]]
Explanation:
For the points (0,0), (0,2), (2,0), (2,2): floor((100+200+200+50)/4) = floor(137.5) = 137
For the points (0,1), (1,0), (1,2), (2,1): floor((200+200+50+200+100+100)/6) = floor(141.666667) = 141
For the point (1,1): floor((50+200+200+200+200+100+100+100+100)/9) = floor(138.888889) = 138

Constraints:

  • m == img.length
  • n == img[i].length
  • 1 ≤ m, n ≤ 200
  • 0 ≤ img[i][j] ≤ 255

Python Solution

class Solution:
    def imageSmoother(self, img: List[List[int]]) -> List[List[int]]:
        m, n = len(img), len(img[0])
        result = [[0] * n for _ in range(m)]
        
        for i in range(m):
            for j in range(n):
                total = 0
                count = 0
                
                # Check all 9 cells in 3x3 window
                for r in range(max(0, i-1), min(m, i+2)):
                    for c in range(max(0, j-1), min(n, j+2)):
                        total += img[r][c]
                        count += 1
                
                result[i][j] = total // count
        
        return result

Alternative Solution (Using Directions):

class Solution:
    def imageSmoother(self, img: List[List[int]]) -> List[List[int]]:
        m, n = len(img), len(img[0])
        result = [[0] * n for _ in range(m)]
        directions = [(-1,-1), (-1,0), (-1,1), (0,-1), (0,0), (0,1), (1,-1), (1,0), (1,1)]
        
        for i in range(m):
            for j in range(n):
                total = 0
                count = 0
                
                for di, dj in directions:
                    ni, nj = i + di, j + dj
                    if 0 <= ni < m and 0 <= nj < n:
                        total += img[ni][nj]
                        count += 1
                
                result[i][j] = total // count
        
        return result

Implementation Notes:

  • First solution uses nested loops with range limits
  • Second solution uses direction array for cleaner code
  • Both solutions have O(m×n) time complexity

Java Solution

class Solution {
    public int[][] imageSmoother(int[][] img) {
        int m = img.length;
        int n = img[0].length;
        int[][] result = new int[m][n];
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                int total = 0;
                int count = 0;
                
                // Check all 9 cells in 3x3 window
                for (int r = Math.max(0, i-1); r < Math.min(m, i+2); r++) {
                    for (int c = Math.max(0, j-1); c < Math.min(n, j+2); c++) {
                        total += img[r][c];
                        count++;
                    }
                }
                
                result[i][j] = total / count;
            }
        }
        
        return result;
    }
}

JavaScript Solution

/**
 * @param {number[][]} img
 * @return {number[][]}
 */
var imageSmoother = function(img) {
    const m = img.length;
    const n = img[0].length;
    const result = Array(m).fill().map(() => Array(n).fill(0));
    
    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            let total = 0;
            let count = 0;
            
            // Check all 9 cells in 3x3 window
            for (let r = Math.max(0, i-1); r < Math.min(m, i+2); r++) {
                for (let c = Math.max(0, j-1); c < Math.min(n, j+2); c++) {
                    total += img[r][c];
                    count++;
                }
            }
            
            result[i][j] = Math.floor(total / count);
        }
    }
    
    return result;
};

C# Solution

public class Solution {
    public int[][] ImageSmoother(int[][] img) {
        int m = img.Length;
        int n = img[0].Length;
        int[][] result = new int[m][];
        
        for (int i = 0; i < m; i++) {
            result[i] = new int[n];
            for (int j = 0; j < n; j++) {
                int total = 0;
                int count = 0;
                
                // Check all 9 cells in 3x3 window
                for (int r = Math.Max(0, i-1); r < Math.Min(m, i+2); r++) {
                    for (int c = Math.Max(0, j-1); c < Math.Min(n, j+2); c++) {
                        total += img[r][c];
                        count++;
                    }
                }
                
                result[i][j] = total / count;
            }
        }
        
        return result;
    }
}

Implementation Notes:

  • Uses integer division for automatic floor operation
  • Handles edge cases by limiting window boundaries
  • Time complexity: O(m×n), Space complexity: O(m×n)
