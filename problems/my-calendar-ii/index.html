731. My Calendar II

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are implementing a program to use as your calendar. We can add a new event if adding the event will not cause a triple booking.

A triple booking happens when three events have some non-empty intersection (i.e., some moment is common to all three events.).

The event can be represented as a pair of integers start and end that represents a booking on the half-open interval [start, end), the range of real numbers x such that start ≤ x < end.

Implement the MyCalendarTwo class:

  • MyCalendarTwo() Initializes the calendar object.
  • boolean book(int start, int end) Returns true if the event can be added to the calendar successfully without causing a triple booking. Otherwise, return false and do not add the event to the calendar.

Examples:

Example 1:

Input
["MyCalendarTwo", "book", "book", "book", "book", "book", "book"]
[[], [10, 20], [50, 60], [10, 40], [5, 15], [5, 10], [25, 55]]
Output
[null, true, true, true, false, true, true]

Explanation
MyCalendarTwo myCalendarTwo = new MyCalendarTwo();
myCalendarTwo.book(10, 20); // return True, The event can be booked. 
myCalendarTwo.book(50, 60); // return True, The event can be booked. 
myCalendarTwo.book(10, 40); // return True, The third event can be double booked. 
myCalendarTwo.book(5, 15);  // return False, The event cannot be booked, because it would result in a triple booking.
myCalendarTwo.book(5, 10); // return True, The event can be booked, as it does not use time 10 which is already double booked.
myCalendarTwo.book(25, 55); // return True, The event can be booked, as the time in [25, 40) will be double booked with the third event, the time [40, 50) will be single booked, and the time [50, 55) will be double booked with the second event.

Constraints:

  • 0 ≤ start < end ≤ 10^9
  • At most 1000 calls will be made to book.

Python Solution

class MyCalendarTwo:
    def __init__(self):
        self.bookings = []  # List of all bookings
        self.overlaps = []  # List of overlapping intervals
    
    def book(self, start: int, end: int) -> bool:
        # Check if this booking would create a triple booking
        for s, e in self.overlaps:
            if start < e and end > s:
                return False
        
        # Add to overlaps if it overlaps with any existing booking
        for s, e in self.bookings:
            if start < e and end > s:
                self.overlaps.append((max(start, s), min(end, e)))
        
        self.bookings.append((start, end))
        return True

Implementation Notes:

  • Uses two lists to track bookings and overlapping intervals
  • Time complexity: O(n) for each booking where n is the number of existing bookings
  • Space complexity: O(n) to store all bookings and overlaps

Java Solution

class MyCalendarTwo {
    private List bookings;
    private List overlaps;
    
    public MyCalendarTwo() {
        bookings = new ArrayList<>();
        overlaps = new ArrayList<>();
    }
    
    public boolean book(int start, int end) {
        // Check if this booking would create a triple booking
        for (int[] overlap : overlaps) {
            if (start < overlap[1] && end > overlap[0]) {
                return false;
            }
        }
        
        // Add to overlaps if it overlaps with any existing booking
        for (int[] booking : bookings) {
            if (start < booking[1] && end > booking[0]) {
                overlaps.add(new int[]{Math.max(start, booking[0]), Math.min(end, booking[1])});
            }
        }
        
        bookings.add(new int[]{start, end});
        return true;
    }
}

C# Solution

public class MyCalendarTwo {
    private List<(int start, int end)> bookings;
    private List<(int start, int end)> overlaps;
    
    public MyCalendarTwo() {
        bookings = new List<(int start, int end)>();
        overlaps = new List<(int start, int end)>();
    }
    
    public bool Book(int start, int end) {
        // Check if this booking would create a triple booking
        foreach (var overlap in overlaps) {
            if (start < overlap.end && end > overlap.start) {
                return false;
            }
        }
        
        // Add to overlaps if it overlaps with any existing booking
        foreach (var booking in bookings) {
            if (start < booking.end && end > booking.start) {
                overlaps.Add((Math.Max(start, booking.start), Math.Min(end, booking.end)));
            }
        }
        
        bookings.Add((start, end));
        return true;
    }
}

Implementation Notes:

  • Uses two lists to track bookings and overlapping intervals
  • Time complexity: O(n) for each booking where n is the number of existing bookings
  • Space complexity: O(n) to store all bookings and overlaps
