781. Rabbits in Forest

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

There is a forest with an unknown number of rabbits. We asked n rabbits "How many rabbits have the same color as you?" and collected the answers in an integer array answers where answers[i] is the answer of the ith rabbit.

Given the array answers, return the minimum number of rabbits that could be in the forest.

Examples:

Example 1:

Input: answers = [1,1,2]
Output: 5
Explanation:
The two rabbits that answered "1" could both be the same color, say red.
The rabbit that answered "2" can't be red or the answers would be inconsistent.
Say the rabbit that answered "2" was blue.
Then there should be 2 other blue rabbits in the forest that didn't answer into the array.
The smallest possible number of rabbits in the forest is therefore 5: 3 that answered plus 2 that didn't.

Example 2:

Input: answers = [10,10,10]
Output: 11

Constraints:

  • 1 ≤ answers.length ≤ 1000
  • 0 ≤ answers[i] < 1000

Python Solution

class Solution:
    def numRabbits(self, answers: List[int]) -> int:
        count = {}
        for ans in answers:
            count[ans] = count.get(ans, 0) + 1
            
        result = 0
        for ans, freq in count.items():
            # For each answer x, we need at least x+1 rabbits
            # If we have more than x+1 rabbits with the same answer,
            # we need to create new groups
            result += (freq + ans) // (ans + 1) * (ans + 1)
        return result

Implementation Notes:

  • Uses a hash map to count frequencies of answers
  • For each answer x, we need at least x+1 rabbits
  • Time complexity: O(n)
  • Space complexity: O(n)

Java Solution

class Solution {
    public int numRabbits(int[] answers) {
        Map count = new HashMap<>();
        for (int ans : answers) {
            count.put(ans, count.getOrDefault(ans, 0) + 1);
        }
        
        int result = 0;
        for (Map.Entry entry : count.entrySet()) {
            int ans = entry.getKey();
            int freq = entry.getValue();
            // For each answer x, we need at least x+1 rabbits
            // If we have more than x+1 rabbits with the same answer,
            // we need to create new groups
            result += (freq + ans) / (ans + 1) * (ans + 1);
        }
        return result;
    }
}

JavaScript Solution

function numRabbits(answers) {
    const count = {};
    for (const ans of answers) {
        count[ans] = (count[ans] || 0) + 1;
    }
    
    let result = 0;
    for (const [ans, freq] of Object.entries(count)) {
        // For each answer x, we need at least x+1 rabbits
        // If we have more than x+1 rabbits with the same answer,
        // we need to create new groups
        result += Math.ceil(freq / (parseInt(ans) + 1)) * (parseInt(ans) + 1);
    }
    return result;
}

C# Solution

public class Solution {
    public int NumRabbits(int[] answers) {
        Dictionary count = new Dictionary();
        foreach (int ans in answers) {
            if (!count.ContainsKey(ans)) {
                count[ans] = 0;
            }
            count[ans]++;
        }
        
        int result = 0;
        foreach (var entry in count) {
            int ans = entry.Key;
            int freq = entry.Value;
            // For each answer x, we need at least x+1 rabbits
            // If we have more than x+1 rabbits with the same answer,
            // we need to create new groups
            result += (freq + ans) / (ans + 1) * (ans + 1);
        }
        return result;
    }
}

Implementation Notes:

  • Uses a dictionary to count frequencies of answers
  • For each answer x, we need at least x+1 rabbits
  • Time complexity: O(n)
  • Space complexity: O(n)
