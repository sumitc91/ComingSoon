885. Spiral Matrix III

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You start at the cell (rStart, cStart) of an rows x cols grid facing east. The northwest corner is at the first row and column in the grid, and the southeast corner is at the last row and column.

You will walk in a clockwise spiral shape to visit every position in this grid. Whenever you move outside the grid's boundary, we continue our walk outside the grid (but may return to the grid boundary later.). Eventually, we reach all rows * cols spaces of the grid.

Return an array of coordinates representing the positions of the grid in the order you visited them.

Examples:

Example 1:

Input: rows = 1, cols = 4, rStart = 0, cStart = 0
Output: [[0,0],[0,1],[0,2],[0,3]]

Example 2:

Input: rows = 5, cols = 6, rStart = 1, cStart = 4
Output: [[1,4],[1,5],[2,5],[2,4],[2,3],[1,3],[0,3],[0,4],[0,5],[3,5],[3,4],[3,3],[3,2],[2,2],[1,2],[0,2],[4,5],[4,4],[4,3],[4,2],[4,1],[3,1],[2,1],[1,1],[0,1],[4,0],[3,0],[2,0],[1,0],[0,0]]

Constraints:

  • 1 ≤ rows, cols ≤ 100
  • 0 ≤ rStart < rows
  • 0 ≤ cStart < cols

Python Solution

class Solution:
    def spiralMatrixIII(self, rows: int, cols: int, rStart: int, cStart: int) -> List[List[int]]:
        result = [[rStart, cStart]]
        if rows * cols == 1:
            return result
            
        # Direction vectors for east, south, west, north
        directions = [(0, 1), (1, 0), (0, -1), (-1, 0)]
        d = 0  # Start facing east
        step = 0  # Current step length
        r, c = rStart, cStart
        
        while len(result) < rows * cols:
            if d % 2 == 0:
                step += 1  # Increase step when moving east or west
                
            # Move in current direction
            for _ in range(step):
                r += directions[d][0]
                c += directions[d][1]
                
                # Add position if it's within bounds
                if 0 <= r < rows and 0 <= c < cols:
                    result.append([r, c])
                    
            # Turn 90 degrees clockwise
            d = (d + 1) % 4
            
        return result

Implementation Notes:

  • Uses direction vectors for movement
  • Time complexity: O(max(rows, cols)²)
  • Space complexity: O(rows * cols)

Java Solution

class Solution {
    public int[][] spiralMatrixIII(int rows, int cols, int rStart, int cStart) {
        int[][] result = new int[rows * cols][2];
        int index = 0;
        result[index++] = new int[]{rStart, cStart};
        
        if (rows * cols == 1) return result;
        
        // Direction vectors for east, south, west, north
        int[][] directions = {{0, 1}, {1, 0}, {0, -1}, {-1, 0}};
        int d = 0;  // Start facing east
        int step = 0;  // Current step length
        int r = rStart, c = cStart;
        
        while (index < rows * cols) {
            if (d % 2 == 0) {
                step++;  // Increase step when moving east or west
            }
            
            // Move in current direction
            for (int i = 0; i < step; i++) {
                r += directions[d][0];
                c += directions[d][1];
                
                // Add position if it's within bounds
                if (r >= 0 && r < rows && c >= 0 && c < cols) {
                    result[index++] = new int[]{r, c};
                }
            }
            
            // Turn 90 degrees clockwise
            d = (d + 1) % 4;
        }
        
        return result;
    }
}

C# Solution

public class Solution {
    public int[][] SpiralMatrixIII(int rows, int cols, int rStart, int cStart) {
        var result = new List { new int[] { rStart, cStart } };
        if (rows * cols == 1) return result.ToArray();
        
        // Direction vectors for east, south, west, north
        var directions = new (int, int)[] { (0, 1), (1, 0), (0, -1), (-1, 0) };
        int d = 0;  // Start facing east
        int step = 0;  // Current step length
        int r = rStart, c = cStart;
        
        while (result.Count < rows * cols) {
            if (d % 2 == 0) {
                step++;  // Increase step when moving east or west
            }
            
            // Move in current direction
            for (int i = 0; i < step; i++) {
                r += directions[d].Item1;
                c += directions[d].Item2;
                
                // Add position if it's within bounds
                if (r >= 0 && r < rows && c >= 0 && c < cols) {
                    result.Add(new int[] { r, c });
                }
            }
            
            // Turn 90 degrees clockwise
            d = (d + 1) % 4;
        }
        
        return result.ToArray();
    }
}

Implementation Notes:

  • Uses tuples for direction vectors in C#
  • Handles boundary checking efficiently
  • Maintains clockwise spiral pattern
