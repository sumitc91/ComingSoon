789. Escape The Ghosts

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are playing a simplified Pacman game. You start at the point (0, 0), and your goal is to travel to the target: (target[0], target[1]). There are several ghosts on the map, the i-th ghost starts at (ghosts[i][0], ghosts[i][1]).

Each turn, you and all ghosts simultaneously may move in one of 4 cardinal directions: north, east, west, or south, going from the previous point to a new point 1 unit of distance away.

You escape if and only if you can reach the target before any ghost reaches you (for any given moves the ghosts may take.) If you reach any square (including the target) at the same time as a ghost, it doesn't count as an escape.

Return true if and only if it is possible to escape.

Examples:

Example 1:

Input: ghosts = [[1,0],[0,3]], target = [0,1]
Output: true
Explanation: You can reach the destination (0, 1) before any ghost reaches you.

Example 2:

Input: ghosts = [[1,0]], target = [2,0]
Output: false
Explanation: You need to reach the destination (2, 0), but the ghost at (1, 0) lies between your destination and you.

Example 3:

Input: ghosts = [[2,0]], target = [1,0]
Output: false
Explanation: The ghost can reach the target at the same time as you.

Constraints:

  • 1 ≤ ghosts.length ≤ 100
  • ghosts[i].length == 2
  • -10^4 ≤ ghosts[i][j] ≤ 10^4
  • target.length == 2
  • -10^4 ≤ target[j] ≤ 10^4

Python Solution

class Solution:
    def escapeGhosts(self, ghosts: List[List[int]], target: List[int]) -> bool:
        def manhattan_distance(x1, y1, x2, y2):
            return abs(x1 - x2) + abs(y1 - y2)
            
        # Calculate distance from player to target
        player_distance = manhattan_distance(0, 0, target[0], target[1])
        
        # Check if any ghost can reach target before or at same time as player
        for ghost in ghosts:
            ghost_distance = manhattan_distance(ghost[0], ghost[1], target[0], target[1])
            if ghost_distance <= player_distance:
                return False
                
        return True

Implementation Notes:

  • Uses Manhattan distance to calculate shortest path
  • Checks if any ghost can reach target before or at same time as player
  • Time complexity: O(n) where n is number of ghosts
  • Space complexity: O(1)

Java Solution

class Solution {
    public boolean escapeGhosts(int[][] ghosts, int[] target) {
        int playerDistance = manhattanDistance(0, 0, target[0], target[1]);
        
        for (int[] ghost : ghosts) {
            int ghostDistance = manhattanDistance(ghost[0], ghost[1], target[0], target[1]);
            if (ghostDistance <= playerDistance) {
                return false;
            }
        }
        
        return true;
    }
    
    private int manhattanDistance(int x1, int y1, int x2, int y2) {
        return Math.abs(x1 - x2) + Math.abs(y1 - y2);
    }
}
