937. Reorder Data in Log Files

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an array of logs. Each log is a space-delimited string of words, where the first word is the identifier.

There are two types of logs:

  • Letter-logs: All words (except the identifier) consist of lowercase English letters.
  • Digit-logs: All words (except the identifier) consist of digits.

Reorder these logs so that:

  1. The letter-logs come before all digit-logs.
  2. The letter-logs are sorted lexicographically by their contents. If their contents are the same, then sort them lexicographically by their identifiers.
  3. The digit-logs maintain their relative ordering.

Return the final order of the logs.

Example 1:

Input: logs = ["dig1 8 1 5 1","let1 art can","dig2 3 6","let2 own kit dig","let3 art zero"]
Output: ["let1 art can","let3 art zero","let2 own kit dig","dig1 8 1 5 1","dig2 3 6"]
Explanation:
The letter-log contents are all different, so their ordering is "art can", "art zero", "own kit dig".
The digit-logs have a relative order of "dig1 8 1 5 1", "dig2 3 6".

Example 2:

Input: logs = ["a1 9 2 3 1","g1 act car","zo4 4 7","ab1 off key dog","a8 act zoo"]
Output: ["g1 act car","a8 act zoo","ab1 off key dog","a1 9 2 3 1","zo4 4 7"]

Constraints:

  • 1 <= logs.length <= 100
  • 3 <= logs[i].length <= 100
  • All tokens in logs[i] are separated by a single space.
  • logs[i] is guaranteed to have an identifier and at least one word after the identifier.

Solution

Python Solution

class Solution:
    def reorderLogFiles(self, logs: List[str]) -> List[str]:
        def get_key(log):
            id_, rest = log.split(" ", 1)
            return (0, rest, id_) if rest[0].isalpha() else (1,)
        
        return sorted(logs, key=get_key)

Time Complexity: O(n * log n)

Where n is the number of logs. The sorting operation takes O(n * log n) time.

Space Complexity: O(n)

We need to store the sorted logs.

Java Solution

class Solution {
    public String[] reorderLogFiles(String[] logs) {
        Arrays.sort(logs, (log1, log2) -> {
            String[] split1 = log1.split(" ", 2);
            String[] split2 = log2.split(" ", 2);
            boolean isDigit1 = Character.isDigit(split1[1].charAt(0));
            boolean isDigit2 = Character.isDigit(split2[1].charAt(0));
            
            if (!isDigit1 && !isDigit2) {
                int cmp = split1[1].compareTo(split2[1]);
                if (cmp != 0) return cmp;
                return split1[0].compareTo(split2[0]);
            }
            return isDigit1 ? (isDigit2 ? 0 : 1) : -1;
        });
        return logs;
    }
}

C++ Solution

class Solution {
public:
    vector reorderLogFiles(vector& logs) {
        auto comp = [](const string& log1, const string& log2) {
            int space1 = log1.find(' ');
            int space2 = log2.find(' ');
            string id1 = log1.substr(0, space1);
            string id2 = log2.substr(0, space2);
            string content1 = log1.substr(space1 + 1);
            string content2 = log2.substr(space2 + 1);
            
            bool isDigit1 = isdigit(content1[0]);
            bool isDigit2 = isdigit(content2[0]);
            
            if (!isDigit1 && !isDigit2) {
                if (content1 != content2) return content1 < content2;
                return id1 < id2;
            }
            return isDigit1 ? (isDigit2 ? false : false) : true;
        };
        
        stable_sort(logs.begin(), logs.end(), comp);
        return logs;
    }
};

JavaScript Solution

/**
 * @param {string[]} logs
 * @return {string[]}
 */
var reorderLogFiles = function(logs) {
    return logs.sort((a, b) => {
        const [id1, ...content1] = a.split(' ');
        const [id2, ...content2] = b.split(' ');
        const isDigit1 = !isNaN(content1[0]);
        const isDigit2 = !isNaN(content2[0]);
        
        if (!isDigit1 && !isDigit2) {
            const cmp = content1.join(' ').localeCompare(content2.join(' '));
            return cmp !== 0 ? cmp : id1.localeCompare(id2);
        }
        return isDigit1 ? (isDigit2 ? 0 : 1) : -1;
    });
};
