800. Similar RGB Color

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

The RGB color "#AABBCC" can be written as "#ABC" in shorthand. For example, "#15c" is shorthand for the color "#1155cc".

Given a string color that follows the format "#ABCDEF", return a string representing the color that is most similar to the given color and has a shorthand (i.e., it can be represented as some "#XYZ").

Any answer which has the same highest similarity as the best answer will be accepted.

For example, the similarity between "123" and "111" is -(1-1)² -(2-1)² -(3-1)² = -5.

Examples:

Example 1:

Input: color = "#09f166"
Output: "#11ee66"
Explanation: 
The similarity is -(0x09-0x11)² -(0xf1-0xee)² -(0x66-0x66)² = -1296 -9 -0 = -1305

Example 2:

Input: color = "#4e3fe1"
Output: "#4444ee"

Constraints:

  • color.length == 7
  • color[0] == '#'
  • color[i] is either digit or character in the range ['a', 'f'] for i > 0

Python Solution

class Solution:
    def similarRGB(self, color: str) -> str:
        def find_closest(comp):
            num = int(comp, 16)
            q = num // 17
            if num % 17 > 8:
                q += 1
            return '{:02x}'.format(17 * q)
        
        return '#' + find_closest(color[1:3]) + find_closest(color[3:5]) + find_closest(color[5:7])

Implementation Notes:

  • Divides each component by 17 to find closest repeating hex value
  • Rounds up if remainder is greater than 8
  • Time complexity: O(1)
  • Space complexity: O(1)

Java Solution

class Solution {
    public String similarRGB(String color) {
        return "#" + findClosest(color.substring(1, 3)) 
                   + findClosest(color.substring(3, 5)) 
                   + findClosest(color.substring(5, 7));
    }
    
    private String findClosest(String comp) {
        int num = Integer.parseInt(comp, 16);
        int q = num / 17;
        if (num % 17 > 8) q++;
        return String.format("%02x", 17 * q);
    }
}

C++ Solution

class Solution {
public:
    string similarRGB(string color) {
        return "#" + findClosest(color.substr(1, 2)) 
                  + findClosest(color.substr(3, 2)) 
                  + findClosest(color.substr(5, 2));
    }
    
private:
    string findClosest(string comp) {
        int num = stoi(comp, nullptr, 16);
        int q = num / 17;
        if (num % 17 > 8) q++;
        
        stringstream ss;
        ss << setfill('0') << setw(2) << hex << 17 * q;
        return ss.str();
    }
};

JavaScript Solution

/**
 * @param {string} color
 * @return {string}
 */
var similarRGB = function(color) {
    const findClosest = (comp) => {
        const num = parseInt(comp, 16);
        let q = Math.floor(num / 17);
        if (num % 17 > 8) q++;
        return (17 * q).toString(16).padStart(2, '0');
    };
    
    return '#' + findClosest(color.slice(1, 3)) 
               + findClosest(color.slice(3, 5)) 
               + findClosest(color.slice(5, 7));
};

C# Solution

public class Solution {
    public string SimilarRGB(string color) {
        return "#" + FindClosest(color.Substring(1, 2)) 
                  + FindClosest(color.Substring(3, 2)) 
                  + FindClosest(color.Substring(5, 2));
    }
    
    private string FindClosest(string comp) {
        int num = Convert.ToInt32(comp, 16);
        int q = num / 17;
        if (num % 17 > 8) q++;
        return (17 * q).ToString("x2");
    }
}

Implementation Notes:

  • Processes each color component (R,G,B) separately
  • Finds closest multiple of 17 for each component
  • Time complexity: O(1)
  • Space complexity: O(1)
