842. Split Array into Fibonacci Sequence

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given a string of digits num, such as "123456579". We can split it into a Fibonacci-like sequence [123, 456, 579].

Formally, a Fibonacci-like sequence is a list F of non-negative integers such that:

  • 0 ≤ F[i] ≤ 2³¹ - 1, (that is, each integer fits in a 32-bit signed integer type)
  • F.length ≥ 3
  • F[i] + F[i+1] = F[i+2] for all 0 ≤ i < F.length - 2

Additionally, note that when splitting the string into pieces, each piece must not have extra leading zeroes, except if the piece is the number 0 itself.

Return any Fibonacci-like sequence split from num, or return [] if it cannot be done.

Examples:

Example 1:

Input: num = "123456579"
Output: [123,456,579]

Example 2:

Input: num = "11235813"
Output: [1,1,2,3,5,8,13]

Example 3:

Input: num = "112358130"
Output: []
Explanation: The task is impossible.

Constraints:

  • 1 ≤ num.length ≤ 200
  • num contains only digits

Python Solution

class Solution:
    def splitIntoFibonacci(self, num: str) -> List[int]:
        def backtrack(index: int, path: List[int]) -> bool:
            if index == len(num):
                return len(path) >= 3
            
            for i in range(index, len(num)):
                # Check for leading zeros
                if num[index] == '0' and i > index:
                    break
                
                # Get current number
                curr = int(num[index:i + 1])
                if curr > 2**31 - 1:
                    break
                
                # Check Fibonacci property
                if len(path) >= 2 and curr > path[-1] + path[-2]:
                    break
                
                if len(path) <= 1 or curr == path[-1] + path[-2]:
                    path.append(curr)
                    if backtrack(i + 1, path):
                        return True
                    path.pop()
            
            return False
        
        result = []
        backtrack(0, result)
        return result

Implementation Notes:

  • Uses backtracking to try different splits
  • Handles constraints like leading zeros and 32-bit integer limit
  • Time complexity: O(n³) in worst case
  • Space complexity: O(n) for recursion stack

Java Solution

class Solution {
    public List splitIntoFibonacci(String num) {
        List result = new ArrayList<>();
        backtrack(num, 0, result);
        return result;
    }
    
    private boolean backtrack(String num, int index, List path) {
        if (index == num.length()) {
            return path.size() >= 3;
        }
        
        for (int i = index; i < num.length(); i++) {
            // Check for leading zeros
            if (num.charAt(index) == '0' && i > index) {
                break;
            }
            
            // Get current number
            String curr = num.substring(index, i + 1);
            if (curr.length() > 10) {
                break;
            }
            
            long currNum = Long.parseLong(curr);
            if (currNum > Integer.MAX_VALUE) {
                break;
            }
            
            // Check Fibonacci property
            if (path.size() >= 2 && currNum > path.get(path.size()-1) + path.get(path.size()-2)) {
                break;
            }
            
            if (path.size() <= 1 || currNum == path.get(path.size()-1) + path.get(path.size()-2)) {
                path.add((int)currNum);
                if (backtrack(num, i + 1, path)) {
                    return true;
                }
                path.remove(path.size() - 1);
            }
        }
        
        return false;
    }
}

JavaScript Solution

/**
 * @param {string} num
 * @return {number[]}
 */
var splitIntoFibonacci = function(num) {
    const result = [];
    
    const backtrack = (index) => {
        if (index === num.length) {
            return result.length >= 3;
        }
        
        for (let i = index; i < num.length; i++) {
            // Check for leading zeros
            if (num[index] === '0' && i > index) {
                break;
            }
            
            // Get current number
            const curr = num.substring(index, i + 1);
            const currNum = parseInt(curr);
            
            if (currNum > 2**31 - 1) {
                break;
            }
            
            // Check Fibonacci property
            if (result.length >= 2 && currNum > result[result.length-1] + result[result.length-2]) {
                break;
            }
            
            if (result.length <= 1 || currNum === result[result.length-1] + result[result.length-2]) {
                result.push(currNum);
                if (backtrack(i + 1)) {
                    return true;
                }
                result.pop();
            }
        }
        
        return false;
    };
    
    backtrack(0);
    return result;
};

public class Solution {
    public IList SplitIntoFibonacci(string num) {
        var result = new List();
        Backtrack(num, 0, result);
        return result;
    }
    
    private bool Backtrack(string num, int index, List path) {
        if (index == num.Length) {
            return path.Count >= 3;
        }
        
        for (int i = index; i < num.Length; i++) {
            // Check for leading zeros
            if (num[index] == '0' && i > index) {
                break;
            }
            
            // Get current number
            string curr = num.Substring(index, i - index + 1);
            if (curr.Length > 10) {
                break;
            }
            
            long currNum;
            if (!long.TryParse(curr, out currNum) || currNum > int.MaxValue) {
                break;
            }
            
            // Check Fibonacci property
            if (path.Count >= 2 && currNum > path[path.Count-1] + path[path.Count-2]) {
                break;
            }
            
            if (path.Count <= 1 || currNum == path[path.Count-1] + path[path.Count-2]) {
                path.Add((int)currNum);
                if (Backtrack(num, i + 1, path)) {
                    return true;
                }
                path.RemoveAt(path.Count - 1);
            }
        }
        
        return false;
    }
}
