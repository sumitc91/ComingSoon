969. Pancake Sorting

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an array of integers arr, sort the array by performing a series of pancake flips.

In one pancake flip we do the following steps:

  • Choose an integer k where 1 ≤ k ≤ arr.length
  • Reverse the sub-array arr[0...k-1] (0-indexed)

For example, if arr = [3,2,1,4] and we performed a pancake flip choosing k = 3, we reverse the sub-array [3,2,1], so arr = [1,2,3,4] after the pancake flip at k = 3.

Return an array of the k-values of the pancake flips that should be performed in order to sort arr.

Example 1:

Input: arr = [3,2,4,1]
Output: [4,2,4,3]
Explanation: 
We perform 4 pancake flips, with k values 4, 2, 4, and 3.
Starting state: arr = [3, 2, 4, 1]
After 1st flip (k = 4): arr = [1, 4, 2, 3]
After 2nd flip (k = 2): arr = [4, 1, 2, 3]
After 3rd flip (k = 4): arr = [3, 2, 1, 4]
After 4th flip (k = 3): arr = [1, 2, 3, 4], which is sorted.
Notice that we return an array of the chosen k values of the pancake flips.

Example 2:

Input: arr = [1,2,3]
Output: []
Explanation: The input is already sorted, so there is no need to flip anything.
Note that other answers, such as [3, 3], would also be accepted.

Constraints:

  • 1 <= arr.length <= 100
  • 1 <= arr[i] <= arr.length
  • All integers in arr are unique (i.e. arr is a permutation of the integers from 1 to arr.length).

Solution

Python Solution

class Solution:
    def pancakeSort(self, arr: List[int]) -> List[int]:
        result = []
        n = len(arr)
        
        for i in range(n, 1, -1):
            max_idx = arr.index(i)
            if max_idx != i - 1:
                if max_idx != 0:
                    result.append(max_idx + 1)
                    arr[:max_idx + 1] = arr[:max_idx + 1][::-1]
                result.append(i)
                arr[:i] = arr[:i][::-1]
        
        return result

Time Complexity: O(n²)

For each position, we need to find the maximum element and perform flips.

Space Complexity: O(1)

We only use a constant amount of extra space.

Java Solution

class Solution {
    public List pancakeSort(int[] arr) {
        List result = new ArrayList<>();
        int n = arr.length;
        
        for (int i = n; i > 1; i--) {
            int maxIdx = 0;
            for (int j = 0; j < i; j++) {
                if (arr[j] > arr[maxIdx]) {
                    maxIdx = j;
                }
            }
            
            if (maxIdx != i - 1) {
                if (maxIdx != 0) {
                    result.add(maxIdx + 1);
                    flip(arr, maxIdx + 1);
                }
                result.add(i);
                flip(arr, i);
            }
        }
        
        return result;
    }
    
    private void flip(int[] arr, int k) {
        int i = 0, j = k - 1;
        while (i < j) {
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;
            i++;
            j--;
        }
    }
}

function flip(arr, k) {
    let i = 0, j = k - 1;
    while (i < j) {
        [arr[i], arr[j]] = [arr[j], arr[i]];
        i++;
        j--;
    }
}

            arr[j] = temp;
            i++;
            j--;
        }
    }
