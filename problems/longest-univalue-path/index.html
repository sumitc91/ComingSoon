687. Longest Univalue Path

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given the root of a binary tree, return the length of the longest path, where each node in the path has the same value. This path may or may not pass through the root.

The length of the path between two nodes is represented by the number of edges between them.

Examples:

Example 1:

Input: root = [5,4,5,1,1,5]
Output: 2
Explanation: The shown image shows that the longest path of the same value (i.e. 5).

Example 2:

Input: root = [1,4,5,4,4,5]
Output: 2
Explanation: The shown image shows that the longest path of the same value (i.e. 4).

Constraints:

  • The number of nodes in the tree is in the range [0, 104]
  • -1000 ≤ Node.val ≤ 1000
  • The depth of the tree will not exceed 1000

Python Solution

class Solution:
    def longestUnivaluePath(self, root: Optional[TreeNode]) -> int:
        self.ans = 0
        
        def arrow_length(node):
            if not node:
                return 0
            
            # Get lengths from children
            left_length = arrow_length(node.left)
            right_length = arrow_length(node.right)
            
            # Initialize lengths for current node
            left_arrow = right_arrow = 0
            
            # Check if left child has same value
            if node.left and node.left.val == node.val:
                left_arrow = left_length + 1
                
            # Check if right child has same value
            if node.right and node.right.val == node.val:
                right_arrow = right_length + 1
                
            # Update global answer
            self.ans = max(self.ans, left_arrow + right_arrow)
            
            # Return maximum arrow length for parent
            return max(left_arrow, right_arrow)
        
        arrow_length(root)
        return self.ans

Implementation Notes:

  • Uses depth-first search with recursion
  • Maintains global answer variable for maximum path length
  • For each node, calculates arrow lengths from left and right children
  • Time complexity: O(N), where N is number of nodes
  • Space complexity: O(H), where H is height of tree

Java Solution

class Solution {
    int ans;
    
    public int longestUnivaluePath(TreeNode root) {
        ans = 0;
        arrowLength(root);
        return ans;
    }
    
    private int arrowLength(TreeNode node) {
        if (node == null) return 0;
        
        int leftLength = arrowLength(node.left);
        int rightLength = arrowLength(node.right);
        
        int leftArrow = 0, rightArrow = 0;
        
        if (node.left != null && node.left.val == node.val) {
            leftArrow = leftLength + 1;
        }
        if (node.right != null && node.right.val == node.val) {
            rightArrow = rightLength + 1;
        }
        
        ans = Math.max(ans, leftArrow + rightArrow);
        return Math.max(leftArrow, rightArrow);
    }
}

        ArrowLength(root);
        return ans;

        int leftLength = ArrowLength(node.left);
        int rightLength = ArrowLength(node.right);

        if (node.left != null && node.left.val == node.val) {
            leftArrow = leftLength + 1;
        }
        if (node.right != null && node.right.val == node.val) {
            rightArrow = rightLength + 1;

        ans = Math.Max(ans, leftArrow + rightArrow);
        return Math.Max(leftArrow, rightArrow);

}

Implementation Notes:

  • Uses depth-first search with recursion
  • Maintains global answer variable for maximum path length
  • For each node, calculates arrow lengths from left and right children
  • Updates answer with sum of left and right arrows
  • Returns maximum arrow length for parent node
  • Time complexity: O(N), where N is number of nodes
  • Space complexity: O(H), where H is height of tree
