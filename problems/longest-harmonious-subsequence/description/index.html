594. Longest Harmonious Subsequence

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

We define a harmonious array as an array where the difference between its maximum value and its minimum value is exactly 1.

Given an integer array nums, return the length of its longest harmonious subsequence among all its possible subsequences.

A subsequence of array is a sequence that can be derived from the array by deleting some or no elements without changing the order of the remaining elements.

Example 1:

Input: nums = [1,3,2,2,5,2,3,7]
Output: 5
Explanation: The longest harmonious subsequence is [3,2,2,2,3].

Example 2:

Input: nums = [1,2,3,4]
Output: 2

Example 3:

Input: nums = [1,1,1,1]
Output: 0

Constraints:

  • 1 <= nums.length <= 2 * 104
  • -109 <= nums[i] <= 109

Solution

Python Solution

class Solution:
    def findLHS(self, nums: List[int]) -> int:
        count = Counter(nums)
        max_length = 0
        
        for num in count:
            if num + 1 in count:
                max_length = max(max_length, count[num] + count[num + 1])
                
        return max_length

Time Complexity: O(n)

Where n is the length of the input array.

Space Complexity: O(n)

For storing the frequency count of numbers.

Java Solution

class Solution {
    public int findLHS(int[] nums) {
        Map count = new HashMap<>();
        for (int num : nums) {
            count.put(num, count.getOrDefault(num, 0) + 1);
        }
        
        int maxLength = 0;
        for (int num : count.keySet()) {
            if (count.containsKey(num + 1)) {
                maxLength = Math.max(maxLength, count.get(num) + count.get(num + 1));
            }
        }
        
        return maxLength;
    }
}

C++ Solution

class Solution {
public:
    int findLHS(vector& nums) {
        unordered_map count;
        for (int num : nums) {
            count[num]++;
        }
        
        int maxLength = 0;
        for (const auto& pair : count) {
            int num = pair.first;
            if (count.find(num + 1) != count.end()) {
                maxLength = max(maxLength, count[num] + count[num + 1]);
            }
        }
        
        return maxLength;
    }
};

var findLHS = function(nums) {
    const count = new Map();
    for (const num of nums) {
        count.set(num, (count.get(num) || 0) + 1);
    }
    
    let maxLength = 0;
    for (const [num, freq] of count) {
        if (count.has(num + 1)) {
            maxLength = Math.max(maxLength, freq + count.get(num + 1));
        }
    }
    
    return maxLength;
};

C# Solution

public class Solution {
    public int FindLHS(int[] nums) {
        var count = new Dictionary();
        foreach (int num in nums) {
            if (!count.ContainsKey(num)) {
                count[num] = 0;
            }
            count[num]++;
        }
        
        int maxLength = 0;
        foreach (int num in count.Keys) {
            if (count.ContainsKey(num + 1)) {
                maxLength = Math.Max(maxLength, count[num] + count[num + 1]);
            }
        }
        
        return maxLength;
    }
}
