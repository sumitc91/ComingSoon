779. K-th Symbol in Grammar

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

We build a table of n rows (1-indexed). We start by writing 0 in the 1st row. Now in every subsequent row, we look at the previous row and replace each occurrence of 0 with 01, and each occurrence of 1 with 10.

For example, for n = 3, the 1st row is 0, the 2nd row is 01, and the 3rd row is 0110.

Given two integer n and k, return the kth (1-indexed) symbol in the nth row of a table of n rows.

Examples:

Example 1:

Input: n = 1, k = 1
Output: 0
Explanation: row 1: 0

Example 2:

Input: n = 2, k = 1
Output: 0
Explanation: 
row 1: 0
row 2: 01

Example 3:

Input: n = 2, k = 2
Output: 1
Explanation: 
row 1: 0
row 2: 01

Constraints:

  • 1 ≤ n ≤ 30
  • 1 ≤ k ≤ 2^n - 1

Python Solution

class Solution:
    def kthGrammar(self, n: int, k: int) -> int:
        if n == 1:
            return 0
        
        # If k is in the first half, it's the same as the previous row
        # If k is in the second half, it's the opposite of the previous row
        length = 2 ** (n - 1)
        if k <= length:
            return self.kthGrammar(n - 1, k)
        else:
            return 1 - self.kthGrammar(n - 1, k - length)

Implementation Notes:

  • Uses recursion to find the k-th symbol
  • Key insight: each row is twice the length of the previous row
  • Time complexity: O(n)
  • Space complexity: O(n) for recursion stack
