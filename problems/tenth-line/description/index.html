195. Tenth Line

Easy

Problem Description

Given a text file file.txt, print just the 10th line of the file.

Examples

Assume that file.txt has the following content:

Line 1
Line 2
Line 3
Line 4
Line 5
Line 6
Line 7
Line 8
Line 9
Line 10

Your script should output the tenth line, which is:

Line 10

Bash Solution


# Using sed
sed -n '10p' file.txt

Alternative Solutions:


# Using awk
awk 'NR == 10' file.txt

# Using head and tail
head -n 10 file.txt | tail -n 1

# Using tail and head
tail -n+10 file.txt | head -n 1

Solution Explanation

There are several ways to solve this problem:

  • sed -n '10p':
    • -n suppresses automatic printing
    • 10p prints only the 10th line
    • Most concise solution
  • awk 'NR == 10':
    • NR is the current line number
    • Prints line when NR equals 10
    • Clean and readable
  • head -n 10 | tail -n 1:
    • Get first 10 lines
    • Then get last line
    • More intuitive approach
  • tail -n+10 | head -n 1:
    • Start from 10th line
    • Get first line of result
    • Alternative pipeline
