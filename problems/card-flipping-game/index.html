822. Card Flipping Game

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given two integer arrays fronts and backs where fronts[i] and backs[i] represent the number on the front and back of the ith card.

Return the smallest number that is on the front of a card and not on the back of any card. If there is no such number, return 0.

Examples:

Example 1:

Input: fronts = [1,2,4,4,7], backs = [1,3,4,1,3]
Output: 2
Explanation: If we flip the second card, the fronts are [1,3,4,4,7] and the backs are [1,2,4,1,3].
2 is the smallest number that is on the front of a card and not on the back of any card.

Example 2:

Input: fronts = [1], backs = [1]
Output: 0

Constraints:

  • n == fronts.length == backs.length
  • 1 ≤ n ≤ 1000
  • 1 ≤ fronts[i], backs[i] ≤ 2000

Python Solution

class Solution:
    def flipgame(self, fronts: List[int], backs: List[int]) -> int:
        # Numbers that appear on both sides of the same card
        same = {x for i, x in enumerate(fronts) if x == backs[i]}
        
        # Find minimum valid number
        ans = float('inf')
        for x in fronts + backs:
            if x not in same:
                ans = min(ans, x)
                
        return ans if ans < float('inf') else 0

Implementation Notes:

  • Identifies numbers that appear on both sides of same card
  • Checks all numbers for valid candidates
  • Time complexity: O(n)
  • Space complexity: O(n)

Java Solution

class Solution {
    public int flipgame(int[] fronts, int[] backs) {
        // Numbers that appear on both sides of the same card
        Set same = new HashSet<>();
        for (int i = 0; i < fronts.length; i++) {
            if (fronts[i] == backs[i]) {
                same.add(fronts[i]);
            }
        }
        
        // Find minimum valid number
        int ans = Integer.MAX_VALUE;
        for (int x : fronts) {
            if (!same.contains(x)) {
                ans = Math.min(ans, x);
            }
        }
        for (int x : backs) {
            if (!same.contains(x)) {
                ans = Math.min(ans, x);
            }
        }
        
        return ans == Integer.MAX_VALUE ? 0 : ans;
    }
}

public class Solution {
    public int Flipgame(int[] fronts, int[] backs) {
        // Numbers that appear on both sides of the same card
        var same = new HashSet();
        for (int i = 0; i < fronts.Length; i++) {
            if (fronts[i] == backs[i]) {
                same.Add(fronts[i]);
            }
        }
        
        // Find minimum valid number
        int ans = int.MaxValue;
        foreach (int x in fronts) {
            if (!same.Contains(x)) {
                ans = Math.Min(ans, x);
            }
        }
        foreach (int x in backs) {
            if (!same.Contains(x)) {
                ans = Math.Min(ans, x);
            }
        }
        
        return ans == int.MaxValue ? 0 : ans;
    }
}

Implementation Notes:

  • Uses HashSet to track numbers appearing on both sides
  • Checks all numbers in both arrays
  • Time complexity: O(n)
  • Space complexity: O(n)
