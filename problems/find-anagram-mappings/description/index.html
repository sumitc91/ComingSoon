760. Find Anagram Mappings

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given two lists A and B, and B is an anagram of A. B is an anagram of A means B is made by randomizing the order of the elements in A.

We want to find an index mapping P, from A to B. A mapping P[i] = j means the ith element in A appears in B at index j.

These lists A and B may contain duplicates. If there are multiple answers, output any of them.

Examples:

Example 1:

Input: A = [12, 28, 46, 32, 50], B = [50, 12, 32, 46, 28]
Output: [1, 4, 3, 2, 0]
Explanation:
As P[0] = 1 because the 0th element of A appears at B[1], and P[1] = 4 because the 1st element of A appears at B[4], and so on.

Constraints:

  • 1 ≤ A.length = B.length ≤ 100
  • A[i], B[i] are integers in range [0, 10^5]

Python Solution

class Solution:
    def anagramMappings(self, A: List[int], B: List[int]) -> List[int]:
        # Create a dictionary mapping each value in B to its index
        d = {x: i for i, x in enumerate(B)}
        
        # Map each value in A to its corresponding index in B
        return [d[x] for x in A]

Implementation Notes:

  • Uses a dictionary to store value-to-index mapping for B
  • Maps each value in A to its corresponding index in B
  • Time complexity: O(n) where n is length of A/B
  • Space complexity: O(n) for the dictionary

Java Solution

class Solution {
    public int[] anagramMappings(int[] A, int[] B) {
        // Create a map to store value-to-index mapping for B
        Map map = new HashMap<>();
        for (int i = 0; i < B.length; i++) {
            map.put(B[i], i);
        }
        
        // Map each value in A to its corresponding index in B
        int[] result = new int[A.length];
        for (int i = 0; i < A.length; i++) {
            result[i] = map.get(A[i]);
        }
        
        return result;
    }
}

JavaScript Solution

function anagramMappings(A, B) {
    // Create a map to store value-to-index mapping for B
    const map = new Map();
    for (let i = 0; i < B.length; i++) {
        map.set(B[i], i);

}

C# Solution

public class Solution {
    public int[] AnagramMappings(int[] A, int[] B) {
        // Create a dictionary to store value-to-index mapping for B
        var map = new Dictionary();
        for (int i = 0; i < B.Length; i++) {
            map[B[i]] = i;
        }
        
        // Map each value in A to its corresponding index in B
        int[] result = new int[A.Length];
        for (int i = 0; i < A.Length; i++) {
            result[i] = map[A[i]];
        }
        
        return result;
    }
}

Implementation Notes:

  • Uses a dictionary/map to store value-to-index mapping for B
  • Maps each value in A to its corresponding index in B
  • Time complexity: O(n) where n is length of A/B
  • Space complexity: O(n) for the dictionary
