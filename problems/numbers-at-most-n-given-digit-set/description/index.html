902. Numbers At Most N Given Digit Set

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an array of digits which is sorted in non-decreasing order. You can write numbers using each digits[i] as many times as we want. For example, if digits = ['1','3','5'], we may write numbers such as '13', '551', and '1351315'.

Return the number of positive integers that can be generated that are less than or equal to a given integer n.

Example 1:

Input: digits = ["1","3","5","7"], n = 100
Output: 20
Explanation: 
The 20 numbers that can be written are:
1, 3, 5, 7, 11, 13, 15, 17, 31, 33, 35, 37, 51, 53, 55, 57, 71, 73, 75, 77.

Example 2:

Input: digits = ["1","4","9"], n = 1000000000
Output: 29523
Explanation: 
We can write 3 one digit numbers, 9 two digit numbers, 27 three digit numbers,
81 four digit numbers, 243 five digit numbers, 729 six digit numbers,
2187 seven digit numbers, 6561 eight digit numbers, and 19683 nine digit numbers.
In total, this is 29523 integers that can be written using the digits array.

Example 3:

Input: digits = ["7"], n = 8
Output: 1

Constraints:

  • 1 <= digits.length <= 9
  • digits[i].length == 1
  • digits[i] is a digit from '1' to '9'.
  • digits are sorted in non-decreasing order.
  • 1 <= n <= 10â¹

Solution

Python Solution

class Solution:
    def atMostNGivenDigitSet(self, digits: List[str], n: int) -> int:
        n_str = str(n)
        n_len = len(n_str)
        digits = [int(d) for d in digits]
        
        # Count numbers with length less than n
        result = sum(len(digits) ** i for i in range(1, n_len))
        
        # Count numbers with same length as n
        for i in range(n_len):
            prefix = n_str[:i]
            current = int(n_str[i])
            
            # Count numbers that start with prefix and have a smaller digit
            for d in digits:
                if d < current:
                    result += len(digits) ** (n_len - i - 1)
            
            # If current digit is not in our set, we can't continue
            if current not in digits:
                return result
                
            # If this is the last digit and it's in our set, add 1
            if i == n_len - 1:
                result += 1
                
        return result

Java Solution

class Solution {
    public int atMostNGivenDigitSet(String[] digits, int n) {
        String nStr = String.valueOf(n);
        int nLen = nStr.length();
        int[] digitArray = new int[digits.length];
        for (int i = 0; i < digits.length; i++) {
            digitArray[i] = Integer.parseInt(digits[i]);
        }
        
        // Count numbers with length less than n
        int result = 0;
        for (int i = 1; i < nLen; i++) {
            result += Math.pow(digits.length, i);
        }
        
        // Count numbers with same length as n
        for (int i = 0; i < nLen; i++) {
            String prefix = nStr.substring(0, i);
            int current = nStr.charAt(i) - '0';
            
            // Count numbers that start with prefix and have a smaller digit
            for (int d : digitArray) {
                if (d < current) {
                    result += Math.pow(digits.length, nLen - i - 1);
                }
            }
            
            // If current digit is not in our set, we can't continue
            boolean found = false;
            for (int d : digitArray) {
                if (d == current) {
                    found = true;
                    break;
                }
            }
            if (!found) {
                return result;
            }
            
            // If this is the last digit and it's in our set, add 1
            if (i == nLen - 1) {
                result++;
            }
        }
        
        return result;
    }
}
