Find the Index of the First Occurrence in a String

Problem Statement: Given two strings, haystack and needle, return the index of the first occurrence of needle in haystack. If needle is not part of haystack, return -1.

Solution (JavaScript):


function strStr(haystack, needle) {
  return haystack.indexOf(needle);
}

Solution (Python):


def strStr(haystack, needle):
  return haystack.find(needle)
