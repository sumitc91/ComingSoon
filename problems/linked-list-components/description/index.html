817. Linked List Components

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given the head of a linked list containing unique integer values and an integer array nums that is a subset of the linked list values.

Return the number of connected components in nums where two values are connected if they appear consecutively in the linked list.

Examples:

Example 1:

Input: head = [0,1,2,3], nums = [0,1,3]
Output: 2
Explanation: 0 and 1 are connected, so [0, 1] and [3] are the two connected components.

Example 2:

Input: head = [0,1,2,3,4], nums = [0,3,1,4]
Output: 2
Explanation: 0 and 1 are connected, 3 and 4 are connected, so [0, 1] and [3, 4] are the two connected components.

Constraints:

  • The number of nodes in the linked list is n
  • 1 ≤ n ≤ 10^4
  • 0 ≤ Node.val < n
  • All the values Node.val are unique
  • 1 ≤ nums.length ≤ n
  • 0 ≤ nums[i] < n
  • All the values of nums are unique

Python Solution

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def numComponents(self, head: Optional[ListNode], nums: List[int]) -> int:
        nums_set = set(nums)
        count = 0
        is_component = False
        
        while head:
            if head.val in nums_set:
                if not is_component:
                    count += 1
                    is_component = True
            else:
                is_component = False
            head = head.next
            
        return count

Implementation Notes:

  • Uses set for O(1) lookup of values in nums
  • Tracks connected components using a flag
  • Time complexity: O(n) where n is length of linked list
  • Space complexity: O(m) where m is length of nums

Java Solution

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public int numComponents(ListNode head, int[] nums) {
        Set numsSet = new HashSet<>();
        for (int num : nums) {
            numsSet.add(num);
        }
        
        int count = 0;
        boolean isComponent = false;
        
        while (head != null) {
            if (numsSet.contains(head.val)) {
                if (!isComponent) {
                    count++;
                    isComponent = true;
                }
            } else {
                isComponent = false;
            }
            head = head.next;
        }
        
        return count;
    }
}

C++ Solution

    int numComponents(ListNode* head, vector& nums) {
        unordered_set numsSet(nums.begin(), nums.end());

        bool isComponent = false;

                    count++;
                    isComponent = true;
                }
            } else {
                isComponent = false;
            }
            head = head->next;
        }
        
        return count;
    }
};

C# Solution

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     public int val;
 *     public ListNode next;
 *     public ListNode(int val=0, ListNode next=null) {
 *         this.val = val;
 *         this.next = next;
 *     }
 * }
 */
public class Solution {
    public int NumComponents(ListNode head, int[] nums) {
        var numsSet = new HashSet(nums);
        int count = 0;
        bool isComponent = false;
        
        while (head != null) {
            if (numsSet.Contains(head.val)) {
                if (!isComponent) {
                    count++;
                    isComponent = true;
                }
            } else {
                isComponent = false;
            }
            head = head.next;
        }
        
        return count;
    }
}

Implementation Notes:

  • Uses HashSet for efficient value lookup
  • Tracks components using boolean flag
  • Time complexity: O(n)
  • Space complexity: O(m)
