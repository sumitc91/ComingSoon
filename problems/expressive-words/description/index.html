809. Expressive Words

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Sometimes people repeat letters to represent extra feeling. For example:

  • "hello" -> "heeellooo" ("e" and "o" are repeated)
  • "hi" -> "hiiii" ("i" is repeated)

In these strings, we have groups of adjacent letters that are all the same character. For instance, "h", "eee", "ll", "ooo".

We call a group that has 3 or more characters of the same type an expressive group. Notice that "ll" is not an expressive group because it only has 2 characters.

Given a string s and an array of strings words, return the number of words[i] that can be stretched to become s. A word can be stretched to become s if all groups in s are expressive groups and the unstretched version of the word equals the stretched version ignoring character counts.

Examples:

Example 1:

Input: s = "heeellooo", words = ["hello", "hi", "helo"]
Output: 1
Explanation: 
We can stretch "hello" into "heeellooo".
We cannot stretch "hi" into "heeellooo".
We cannot stretch "helo" into "heeellooo".

Example 2:

Input: s = "zzzzzyyyyy", words = ["zzyy","zy","zyy"]
Output: 1

Constraints:

  • 1 ≤ s.length ≤ 100
  • 1 ≤ words.length ≤ 100
  • 1 ≤ words[i].length ≤ 100
  • s and words[i] consist of lowercase English letters

Python Solution

class Solution:
    def expressiveWords(self, s: str, words: List[str]) -> int:
        def get_groups(word):
            groups = []
            counts = []
            prev = ''
            count = 0
            
            for c in word:
                if c != prev:
                    if prev:
                        groups.append(prev)
                        counts.append(count)
                    prev = c
                    count = 1
                else:
                    count += 1
            
            groups.append(prev)
            counts.append(count)
            return groups, counts
        
        s_groups, s_counts = get_groups(s)
        result = 0
        
        for word in words:
            w_groups, w_counts = get_groups(word)
            
            if w_groups != s_groups:
                continue
                
            is_stretchy = True
            for i in range(len(w_counts)):
                if s_counts[i] < w_counts[i] or (s_counts[i] < 3 and s_counts[i] != w_counts[i]):
                    is_stretchy = False
                    break
                    
            if is_stretchy:
                result += 1
                
        return result

Implementation Notes:

  • Groups consecutive characters and counts their occurrences
  • Checks if word groups match s groups and if counts satisfy stretchy conditions
  • Time complexity: O(N * M) where N is length of words and M is max length of a word
  • Space complexity: O(M)

Java Solution

class Solution {
    public int expressiveWords(String s, String[] words) {
        int result = 0;
        
        for (String word : words) {
            if (isStretchy(s, word)) {
                result++;
            }
        }
        
        return result;
    }
    
    private boolean isStretchy(String s, String word) {
        int i = 0, j = 0;
        
        while (i < s.length() && j < word.length()) {
            if (s.charAt(i) != word.charAt(j)) {
                return false;
            }
            
            int s_count = getCount(s, i);
            int w_count = getCount(word, j);
            
            if (s_count < w_count || (s_count < 3 && s_count != w_count)) {
                return false;
            }
            
            i += s_count;
            j += w_count;
        }
        
        return i == s.length() && j == word.length();
    }
    
    private int getCount(String str, int start) {
        int count = 0;
        char c = str.charAt(start);
        
        while (start < str.length() && str.charAt(start) == c) {
            count++;
            start++;
        }
        
        return count;
    }
}

JavaScript Solution

/**
 * @param {string} s
 * @param {string[]} words
 * @return {number}
 */
var expressiveWords = function(s, words) {
    const getGroups = (word) => {
        const groups = [];
        const counts = [];
        let prev = '';
        let count = 0;
        
        for (const c of word) {
            if (c !== prev) {
                if (prev) {
                    groups.push(prev);
                    counts.push(count);
                }
                prev = c;
                count = 1;
            } else {
                count++;
            }
        }
        
        groups.push(prev);
        counts.push(count);
        return [groups, counts];
    };
    
    const [s_groups, s_counts] = getGroups(s);
    let result = 0;
    
    for (const word of words) {
        const [w_groups, w_counts] = getGroups(word);
        
        if (w_groups.length !== s_groups.length) {
            continue;
        }
        
        let isStretchy = true;
        for (let i = 0; i < w_groups.length; i++) {
            if (w_groups[i] !== s_groups[i] || 
                s_counts[i] < w_counts[i] || 
                (s_counts[i] < 3 && s_counts[i] !== w_counts[i])) {
                isStretchy = false;
                break;
            }
        }
        
        if (isStretchy) {
            result++;
        }
    }
    
    return result;
};

C# Solution

public class Solution {
    public int ExpressiveWords(string s, string[] words) {
        int result = 0;
        
        foreach (string word in words) {
            if (IsStretchy(s, word)) {
                result++;
            }
        }
        
        return result;
    }
    
    private bool IsStretchy(string s, string word) {
        int i = 0, j = 0;
        
        while (i < s.Length && j < word.Length) {
            if (s[i] != word[j]) {
                return false;
            }
            
            int sCount = GetCount(s, i);
            int wCount = GetCount(word, j);
            
            if (sCount < wCount || (sCount < 3 && sCount != wCount)) {
                return false;
            }
            
            i += sCount;
            j += wCount;
        }
        
        return i == s.Length && j == word.Length;
    }
    
    private int GetCount(string str, int start) {
        int count = 0;
        char c = str[start];
        
        while (start < str.Length && str[start] == c) {
            count++;
            start++;
        }
        
        return count;
    }
}

Implementation Notes:

  • Uses two-pointer approach to compare characters and their counts
  • Checks if groups can be stretched according to the rules
  • Time complexity: O(N * M) where N is length of words and M is max length of a word
  • Space complexity: O(1)
