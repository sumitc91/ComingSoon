949. Largest Time for Given Digits

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an array of 4 digits, return the largest 24 hour time that can be made.

The smallest 24 hour time is 00:00, and the largest is 23:59. Starting from 00:00, a time is larger if more time has elapsed since midnight.

Return the answer as a string of length 5 in the format "HH:MM". If no valid time can be made, return an empty string.

Example 1:

Input: [1,2,3,4]
Output: "23:41"

Example 2:

Input: [5,5,5,5]
Output: ""

Example 3:

Input: [0,0,0,0]
Output: "00:00"

Example 4:

Input: [0,4,0,0]
Output: "04:00"

Constraints:

  • arr.length == 4
  • 0 <= arr[i] <= 9

Solution

Python Solution

class Solution:
    def largestTimeFromDigits(self, arr: List[int]) -> str:
        # Try all possible permutations
        max_time = -1
        for h1, h2, m1, m2 in itertools.permutations(arr):
            hours = 10 * h1 + h2
            minutes = 10 * m1 + m2
            
            # Check if time is valid
            if 0 <= hours < 24 and 0 <= minutes < 60:
                time = hours * 60 + minutes
                max_time = max(max_time, time)
        
        if max_time == -1:
            return ""
            
        # Convert back to string format
        hours = max_time // 60
        minutes = max_time % 60
        return f"{hours:02d}:{minutes:02d}"

Time Complexity: O(1)

Since we only have 4 digits, there are only 24 possible permutations.

Space Complexity: O(1)

We only use a constant amount of extra space.

Java Solution

class Solution {
    public String largestTimeFromDigits(int[] arr) {
        // Try all possible permutations
        int maxTime = -1;
        for (int i = 0; i < 4; i++) {
            for (int j = 0; j < 4; j++) {
                if (j == i) continue;
                for (int k = 0; k < 4; k++) {
                    if (k == i || k == j) continue;
                    int l = 6 - i - j - k;  // The remaining index
                    
                    int hours = 10 * arr[i] + arr[j];
                    int minutes = 10 * arr[k] + arr[l];
                    
                    // Check if time is valid
                    if (hours < 24 && minutes < 60) {
                        maxTime = Math.max(maxTime, hours * 60 + minutes);
                    }
                }
            }
        }
        
        if (maxTime == -1) {
            return "";
        }
        
        // Convert back to string format
        int hours = maxTime / 60;
        int minutes = maxTime % 60;
        return String.format("%02d:%02d", hours, minutes);
    }
}

var largestTimeFromDigits = function(arr) {
    // Try all possible permutations
    let maxTime = -1;
    for (let i = 0; i < 4; i++) {
        for (let j = 0; j < 4; j++) {
            if (j === i) continue;
            for (let k = 0; k < 4; k++) {
                if (k === i || k === j) continue;
                const l = 6 - i - j - k;  // The remaining index
                
                const hours = 10 * arr[i] + arr[j];

                if (hours < 24 && minutes < 60) {

                }
            }
        }
    }

        return "";
