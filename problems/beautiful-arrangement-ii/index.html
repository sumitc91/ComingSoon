667. Beautiful Arrangement II

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given two integers n and k, construct a list answer that contains n different positive integers ranging from 1 to n and obeys the following requirement:

  • Suppose you take the absolute difference between each adjacent pair of elements in the array. There should be exactly k distinct values among these differences.

Return any valid list that satisfies this requirement.

Examples:

Example 1:

Input: n = 3, k = 1
Output: [1,2,3]
Explanation: The [1,2,3] has three different positive integers ranging from 1 to 3, and the [1,1] has exactly 1 distinct integer: 1

Example 2:

Input: n = 3, k = 2
Output: [1,3,2]
Explanation: The [1,3,2] has three different positive integers ranging from 1 to 3, and the [2,1] has exactly 2 distinct integers: 1 and 2.

Constraints:

  • 1 ≤ k < n ≤ 104

Python Solution

class Solution:
    def constructArray(self, n: int, k: int) -> List[int]:
        # Create array with first n numbers
        answer = list(range(1, n + 1))
        
        # For k differences, we need to swap k-1 pairs
        for i in range(1, k):
            # Reverse subarray to create different differences
            answer[i:] = answer[len(answer)-1:i-1:-1]
        
        return answer

Alternative Solution:

class Solution:
    def constructArray(self, n: int, k: int) -> List[int]:
        answer = []
        left, right = 1, n
        
        # Generate k differences by alternating between min and max
        for i in range(k):
            if i % 2 == 0:
                answer.append(left)
                left += 1
            else:
                answer.append(right)
                right -= 1
        
        # Add remaining numbers in ascending/descending order
        if k % 2 == 0:
            answer.extend(range(right, left - 1, -1))
        else:
            answer.extend(range(left, right + 1))
        
        return answer

Implementation Notes:

  • First solution uses array reversal technique
  • Second solution uses two pointers approach
  • Both solutions have O(n) time complexity

Java Solution

class Solution {
    public int[] constructArray(int n, int k) {
        int[] answer = new int[n];
        int left = 1, right = n;
        int index = 0;
        
        // Generate k differences by alternating between min and max
        for (int i = 0; i < k; i++) {
            if (i % 2 == 0) {
                answer[index++] = left++;
            } else {
                answer[index++] = right--;
            }
        }
        
        // Add remaining numbers in ascending/descending order
        if (k % 2 == 0) {
            while (index < n) {
                answer[index++] = right--;
            }
        } else {
            while (index < n) {
                answer[index++] = left++;
            }
        }
        
        return answer;
    }
}

            while (right >= left) {
                answer.push_back(right--);
            }
        } else {
            while (left <= right) {
                answer.push_back(left++);
            }
        }
        
        return answer;
    }
};

C# Solution

public class Solution {
    public int[] ConstructArray(int n, int k) {
        List answer = new List();
        int left = 1, right = n;
        
        // Generate k differences by alternating between min and max
        for (int i = 0; i < k; i++) {
            if (i % 2 == 0) {
                answer.Add(left++);
            } else {
                answer.Add(right--);
            }
        }
        
        // Add remaining numbers in ascending/descending order
        if (k % 2 == 0) {
            while (right >= left) {
                answer.Add(right--);
            }
        } else {
            while (left <= right) {
                answer.Add(left++);
            }
        }
        
        return answer.ToArray();
    }
}

Implementation Notes:

  • Uses two pointers to generate required differences
  • Time complexity: O(n), Space complexity: O(n)
  • Ensures exactly k distinct differences in adjacent pairs
