892. Surface Area of 3D Shapes

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an n x n grid where you have placed some 1 x 1 x 1 cubes. Each value v = grid[i][j] represents a tower of v cubes placed on top of cell (i, j).

After placing these cubes, you need to calculate the total surface area of the resulting shape.

Examples:

Example 1:

Input: grid = [[1,2],[3,4]]
Output: 34

Example 2:

Input: grid = [[1,1,1],[1,0,1],[1,1,1]]
Output: 32

Example 3:

Input: grid = [[2,2,2],[2,1,2],[2,2,2]]
Output: 46

Constraints:

  • n == grid.length == grid[i].length
  • 1 ≤ n ≤ 50
  • 0 ≤ grid[i][j] ≤ 50

Python Solution

class Solution:
    def surfaceArea(self, grid: List[List[int]]) -> int:
        n = len(grid)
        area = 0
        
        for i in range(n):
            for j in range(n):
                if grid[i][j]:
                    # Add all 6 faces of cubes at this position
                    area += 2 + grid[i][j] * 4
                    
                    # Subtract the faces that are covered
                    # By adjacent cubes in the same column
                    if i > 0:
                        area -= min(grid[i][j], grid[i-1][j]) * 2
                    if j > 0:
                        area -= min(grid[i][j], grid[i][j-1]) * 2
                        
        return area

Implementation Notes:

  • Calculates visible faces
  • Time complexity: O(n²)
  • Space complexity: O(1)

Java Solution

class Solution {
    public int surfaceArea(int[][] grid) {
        int n = grid.length;
        int area = 0;
        
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] > 0) {
                    // Add all 6 faces of cubes at this position
                    area += 2 + grid[i][j] * 4;
                    
                    // Subtract the faces that are covered
                    if (i > 0)
                        area -= Math.min(grid[i][j], grid[i-1][j]) * 2;
                    if (j > 0)
                        area -= Math.min(grid[i][j], grid[i][j-1]) * 2;
                }
            }
        }
        
        return area;
    }
}
