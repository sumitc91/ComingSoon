972. Equal Rational Numbers

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given two strings s and t, each of which represents a non-negative rational number, return true if and only if they represent the same number. The strings may use parentheses to denote the repeating part of the rational number.

A rational number can be represented using up to three parts:

  • An integer part
  • A non-repeating part
  • A repeating part

The number will be represented in one of the following three ways:

  • (e.g., "0", "12")
  • <.> (e.g., "0.5", "2.12", "2.0001")
  • <.><(><)> (e.g., "0.1(6)", "2.0(66)", "2.0(0)")

Example 1:

Input: s = "0.(52)", t = "0.5(25)"
Output: true
Explanation: Because "0.(52)" represents 0.52525252..., and "0.5(25)" represents 0.5252525252..., they represent the same number.

Example 2:

Input: s = "0.1666(6)", t = "0.166(66)"
Output: true

Example 3:

Input: s = "0.9(9)", t = "1."
Output: true
Explanation: "0.9(9)" represents 0.999999999... repeated forever, which equals 1.

Constraints:

  • 1 <= s.length, t.length <= 10000
  • s and t consist of digits, '.', '(', and ')'.
  • It is guaranteed that s and t represent valid rational numbers.

Solution

Python Solution

class Solution:
    def isRationalEqual(self, s: str, t: str) -> bool:
        def convert(s: str) -> float:
            if '(' not in s:
                return float(s)
            
            base, rep = s.split('(')
            rep = rep[:-1]
            base = float(base)
            
            if not rep:
                return base
                
            n = len(rep)
            return base + float(rep) / (10**n - 1)
            
        return abs(convert(s) - convert(t)) < 1e-10

Time Complexity: O(n)

We need to process each character in the input strings.

Space Complexity: O(1)

We only use a constant amount of extra space.

Java Solution

class Solution {
    public boolean isRationalEqual(String s, String t) {
        return Math.abs(convert(s) - convert(t)) < 1e-10;
    }
    
    private double convert(String s) {
        if (!s.contains("(")) {
            return Double.parseDouble(s);
        }
        
        String[] parts = s.split("\\(");
        double base = Double.parseDouble(parts[0]);
        String rep = parts[1].substring(0, parts[1].length() - 1);
        
        if (rep.isEmpty()) {
            return base;
        }
        
        int n = rep.length();
        return base + Double.parseDouble(rep) / (Math.pow(10, n) - 1);
    }
}

    function convert(s) {
        if (!s.includes('(')) {
            return parseFloat(s);
        }
        
        const [base, rep] = s.split('(');
        const repeating = rep.slice(0, -1);
        
        if (!repeating) {
            return parseFloat(base);
        }
        
        const n = repeating.length;
        return parseFloat(base) + parseFloat(repeating) / (Math.pow(10, n) - 1);
    }
    
    return Math.abs(convert(s) - convert(t)) < 1e-10;
};

C# Solution

public class Solution {
    public bool IsRationalEqual(string s, string t) {
        return Math.Abs(Convert(s) - Convert(t)) < 1e-10;
    }
    
    private double Convert(string s) {
        int pos = s.IndexOf('(');
        if (pos == -1) {
            return double.Parse(s);
        }
        
        string basePart = s.Substring(0, pos);
        string rep = s.Substring(pos + 1, s.Length - pos - 2);
        
        if (string.IsNullOrEmpty(rep)) {
            return double.Parse(basePart);
        }
        
        int n = rep.Length;
        return double.Parse(basePart) + double.Parse(rep) / (Math.Pow(10, n) - 1);
    }
}
