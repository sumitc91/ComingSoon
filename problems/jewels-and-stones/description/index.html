771. Jewels and Stones

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You're given strings jewels representing the types of stones that are jewels, and stones representing the stones you have. Each character in stones is a type of stone you have. You want to know how many of the stones you have are also jewels.

Letters are case sensitive, so "a" is considered a different type of stone from "A".

Examples:

Example 1:

Input: jewels = "aA", stones = "aAAbbbb"
Output: 3

Example 2:

Input: jewels = "z", stones = "ZZ"
Output: 0

Constraints:

  • 1 ≤ jewels.length, stones.length ≤ 50
  • jewels and stones consist of only English letters
  • All the characters of jewels are unique

Python Solution

class Solution:
    def numJewelsInStones(self, jewels: str, stones: str) -> int:
        # Convert jewels to set for O(1) lookup
        jewel_set = set(jewels)
        
        # Count stones that are jewels
        return sum(1 for stone in stones if stone in jewel_set)

Implementation Notes:

  • Uses a set for O(1) lookup of jewels
  • Time complexity: O(n) where n is length of stones
  • Space complexity: O(m) where m is length of jewels

Java Solution

class Solution {
    public int numJewelsInStones(String jewels, String stones) {
        // Convert jewels to set for O(1) lookup
        Set jewelSet = new HashSet<>();
        for (char jewel : jewels.toCharArray()) {
            jewelSet.add(jewel);
        }
        
        // Count stones that are jewels
        int count = 0;
        for (char stone : stones.toCharArray()) {
            if (jewelSet.contains(stone)) {
                count++;
            }
        }
        
        return count;
    }
}

JavaScript Solution

function numJewelsInStones(jewels, stones) {
    // Convert jewels to set for O(1) lookup
    const jewelSet = new Set(jewels);
    
    // Count stones that are jewels
    return stones.split('').filter(stone => jewelSet.has(stone)).length;
}

C# Solution

public class Solution {
    public int NumJewelsInStones(string jewels, string stones) {
        // Convert jewels to set for O(1) lookup
        var jewelSet = new HashSet(jewels);
        
        // Count stones that are jewels
        return stones.Count(stone => jewelSet.Contains(stone));
    }
}

Implementation Notes:

  • Uses a set for O(1) lookup of jewels
  • Time complexity: O(n) where n is length of stones
  • Space complexity: O(m) where m is length of jewels
