676. Implement Magic Dictionary

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Design a data structure that is initialized with a list of different words. Provided a string, you should determine if you can change exactly one character in this string to match any word in your data structure.

Implement the MagicDictionary class:

  • MagicDictionary() Initializes the object.
  • void buildDict(String[] dictionary) Sets the data structure with an array of distinct strings dictionary.
  • bool search(String searchWord) Returns true if you can change exactly one character in searchWord to match any string in the data structure, otherwise returns false.

Examples:

Example 1:

Input
["MagicDictionary", "buildDict", "search", "search", "search", "search"]
[[], [["hello","leetcode"]], ["hello"], ["hhllo"], ["hell"], ["leetcoded"]]
Output
[null, null, false, true, false, false]

Explanation
MagicDictionary magicDictionary = new MagicDictionary();
magicDictionary.buildDict(["hello", "leetcode"]);
magicDictionary.search("hello"); // return False
magicDictionary.search("hhllo"); // We can change the second 'h' to 'e' to match "hello" so we return True
magicDictionary.search("hell"); // return False
magicDictionary.search("leetcoded"); // return False

Constraints:

  • 1 ≤ dictionary.length ≤ 100
  • 1 ≤ dictionary[i].length ≤ 100
  • dictionary[i] consists of only lowercase English letters
  • All the strings in dictionary are distinct
  • 1 ≤ searchWord.length ≤ 100
  • searchWord consists of only lowercase English letters
  • buildDict will be called only once before search

Python Solution

class MagicDictionary:
    def __init__(self):
        self.dictionary = []
        
    def buildDict(self, dictionary: List[str]) -> None:
        self.dictionary = dictionary
        
    def search(self, searchWord: str) -> bool:
        for word in self.dictionary:
            if len(word) != len(searchWord):
                continue
            diff = 0
            for i in range(len(word)):
                if word[i] != searchWord[i]:
                    diff += 1
            if diff == 1:
                return True
        return False

Alternative Solution (Using Trie):

class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False
        
class MagicDictionary:
    def __init__(self):
        self.root = TrieNode()
        
    def buildDict(self, dictionary: List[str]) -> None:
        for word in dictionary:
            node = self.root
            for char in word:
                if char not in node.children:
                    node.children[char] = TrieNode()
                node = node.children[char]
            node.is_end = True
            
    def search(self, searchWord: str) -> bool:
        def dfs(node: TrieNode, word: str, diff: int) -> bool:
            if not word:
                return node.is_end and diff == 1
                
            char = word[0]
            if char in node.children:
                if dfs(node.children[char], word[1:], diff):
                    return True
                    
            if diff == 0:
                for child in node.children.values():
                    if child != node.children.get(char):
                        if dfs(child, word[1:], 1):
                            return True
            return False
            
        return dfs(self.root, searchWord, 0)

Implementation Notes:

  • First solution uses simple string comparison
  • Second solution uses Trie for better space efficiency
  • Time complexity: O(n*m) for first solution, O(m) for second
  • Space complexity: O(n*m) for first solution, O(n*m) for second

Java Solution

class MagicDictionary {
    private String[] dictionary;
    
    public MagicDictionary() {
        dictionary = new String[0];
    }
    
    public void buildDict(String[] dictionary) {
        this.dictionary = dictionary;
    }
    
    public boolean search(String searchWord) {
        for (String word : dictionary) {
            if (word.length() != searchWord.length()) {
                continue;
            }
            int diff = 0;
            for (int i = 0; i < word.length(); i++) {
                if (word.charAt(i) != searchWord.charAt(i)) {
                    diff++;
                }
            }
            if (diff == 1) {
                return true;
            }
        }
        return false;
    }
}

    }

        this.dictionary = dictionary;
    }
    
    public bool Search(string searchWord) {
        foreach (string word in dictionary) {
            if (word.Length != searchWord.Length) {
                continue;
            }
            int diff = 0;
            for (int i = 0; i < word.Length; i++) {
                if (word[i] != searchWord[i]) {
                    diff++;
                }
            }
            if (diff == 1) {
                return true;
            }
        }
        return false;
    }
}

Implementation Notes:

  • Simple string comparison approach
  • Time complexity: O(n*m) where n is number of words and m is word length
  • Space complexity: O(n*m)
  • Handles edge cases like different length words
