754. Reach a Number

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are standing at position 0 on an infinite number line. There is a destination at position target.

You can make some number of moves numMoves so that:

  • On each move, you can either go left or right.
  • During the ith move (starting from i == 1 to i == numMoves), you take i steps in the chosen direction.

Given the integer target, return the minimum number of moves required (i.e., the minimum numMoves) to reach the destination.

Examples:

Example 1:

Input: target = 2
Output: 3
Explanation:
On the first move, we step from 0 to 1 (1 step).
On the second move, we step from 1 to -1 (2 steps).
On the third move, we step from -1 to 2 (3 steps).

Example 2:

Input: target = 3
Output: 2
Explanation:
On the first move, we step from 0 to 1 (1 step).
On the second move, we step from 1 to 3 (2 steps).

Constraints:

  • -10^9 ≤ target ≤ 10^9
  • target != 0

Python Solution

class Solution:
    def reachNumber(self, target: int) -> int:
        target = abs(target)
        k = 0
        while target > 0:
            k += 1
            target -= k
            
        # If target is 0, we've reached it
        if target == 0:
            return k
            
        # If target is negative and even, we can reach it
        if target % 2 == 0:
            return k
            
        # If target is negative and odd, we need one more step
        return k + 1 + (k % 2)

Implementation Notes:

  • Uses mathematical approach to find minimum moves
  • First finds k where sum(1 to k) >= target
  • Time complexity: O(sqrt(target))
  • Space complexity: O(1)
