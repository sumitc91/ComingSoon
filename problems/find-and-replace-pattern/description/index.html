890. Find and Replace Pattern

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a list of strings words and a string pattern, return a list of words[i] that match pattern. You may return the answer in any order.

A word matches the pattern if there exists a permutation of letters p so that after replacing every letter x in the pattern with p(x), we get the desired word.

Recall that a permutation of letters is a bijection from letters to letters: every letter maps to another letter, and no two letters map to the same letter.

Examples:

Example 1:

Input: words = ["abc","deq","mee","aqq","dkd","ccc"], pattern = "abb"
Output: ["mee","aqq"]
Explanation: "mee" matches the pattern because there is a permutation {a -> m, b -> e, ...}. 
"ccc" does not match the pattern because {a -> c, b -> c, ...} is not a permutation, since a and b map to the same letter.

Example 2:

Input: words = ["a","b","c"], pattern = "a"
Output: ["a","b","c"]

Constraints:

  • 1 ≤ pattern.length ≤ 20
  • 1 ≤ words.length ≤ 50
  • words[i].length == pattern.length
  • pattern and words[i] are lowercase English letters

Python Solution

class Solution:
    def findAndReplacePattern(self, words: List[str], pattern: str) -> List[str]:
        def match(word):
            if len(word) != len(pattern):
                return False
                
            m1, m2 = {}, {}
            for w, p in zip(word, pattern):
                if w not in m1: m1[w] = p
                if p not in m2: m2[p] = w
                if (m1[w], m2[p]) != (p, w):
                    return False
            return True
            
        return [word for word in words if match(word)]

Implementation Notes:

  • Uses two-way mapping to ensure bijection
  • Time complexity: O(N * K) where N is length of words and K is length of pattern
  • Space complexity: O(K)

Java Solution

class Solution {
    public List findAndReplacePattern(String[] words, String pattern) {
        List result = new ArrayList<>();
        for (String word : words) {
            if (match(word, pattern)) {
                result.add(word);
            }
        }
        return result;
    }
    
    private boolean match(String word, String pattern) {
        Map m1 = new HashMap<>();
        Map m2 = new HashMap<>();
        
        for (int i = 0; i < word.length(); ++i) {
            char w = word.charAt(i);
            char p = pattern.charAt(i);
            
            if (!m1.containsKey(w)) m1.put(w, p);
            if (!m2.containsKey(p)) m2.put(p, w);
            
            if (m1.get(w) != p || m2.get(p) != w)
                return false;
        }
        
        return true;
    }
}

                result.push_back(word);
            }

private:
    bool match(const string& word, const string& pattern) {
        unordered_map m1, m2;
        
        for (int i = 0; i < word.length(); ++i) {
            char w = word[i];
            char p = pattern[i];
            
            if (m1.find(w) == m1.end()) m1[w] = p;
            if (m2.find(p) == m2.end()) m2[p] = w;
            
            if (m1[w] != p || m2[p] != w)
                return false;
        }
        
        return true;
    }
};

JavaScript Solution

/**
 * @param {string[]} words
 * @param {string} pattern
 * @return {string[]}
 */
var findAndReplacePattern = function(words, pattern) {
    const match = (word) => {
        const m1 = new Map();
        const m2 = new Map();
        
        for (let i = 0; i < word.length; i++) {
            const w = word[i];
            const p = pattern[i];
            
            if (!m1.has(w)) m1.set(w, p);
            if (!m2.has(p)) m2.set(p, w);
            
            if (m1.get(w) !== p || m2.get(p) !== w)
                return false;
        }
        
        return true;
    };
    
    return words.filter(word => match(word));
};

C# Solution

public class Solution {
    public IList FindAndReplacePattern(string[] words, string pattern) {
        var result = new List();
        foreach (var word in words) {
            if (Match(word, pattern)) {
                result.Add(word);
            }
        }
        return result;
    }
    
    private bool Match(string word, string pattern) {
        var m1 = new Dictionary();
        var m2 = new Dictionary();
        
        for (int i = 0; i < word.Length; i++) {
            char w = word[i];
            char p = pattern[i];
            
            if (!m1.ContainsKey(w)) m1[w] = p;
            if (!m2.ContainsKey(p)) m2[p] = w;
            
            if (m1[w] != p || m2[p] != w)
                return false;
        }
        
        return true;
    }
}

Implementation Notes:

  • Uses Dictionary for bijective mapping
  • Checks both forward and reverse mappings
  • Efficient character-by-character comparison
