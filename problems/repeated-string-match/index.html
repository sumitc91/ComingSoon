686. Repeated String Match

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given two strings a and b, return the minimum number of times you should repeat string a so that string b is a substring of it. If no such solution exists, return -1.

Note: string a will be repeated multiple times to form a new string, and b should be a substring of this new string.

Examples:

Example 1:

Input: a = "abcd", b = "cdabcdab"
Output: 3
Explanation: We return 3 because by repeating a three times "abcdabcdabcd", b is a substring of it.

Example 2:

Input: a = "a", b = "aa"
Output: 2

Example 3:

Input: a = "abc", b = "wxyz"
Output: -1

Constraints:

  • 1 ≤ a.length ≤ 104
  • 1 ≤ b.length ≤ 104
  • a and b consist of lowercase English letters

Python Solution

class Solution:
    def repeatedStringMatch(self, a: str, b: str) -> int:
        # Calculate minimum repetitions needed
        q = (len(b) - 1) // len(a) + 1
        
        # Try with minimum repetitions
        if b in a * q:
            return q
        
        # Try with one more repetition
        if b in a * (q + 1):
            return q + 1
        
        return -1

Implementation Notes:

  • Calculate minimum repetitions needed based on length of strings
  • Check if b is substring with minimum repetitions
  • If not found, try with one more repetition
  • Time complexity: O(N), where N is length of final string
  • Space complexity: O(N)

Java Solution

class Solution {
    public int repeatedStringMatch(String a, String b) {
        StringBuilder sb = new StringBuilder();
        int count = 0;
        
        // Add string a until length is at least equal to b
        while (sb.length() < b.length()) {
            sb.append(a);
            count++;
        }
        
        // Check if b is substring
        if (sb.toString().contains(b)) {
            return count;
        }
        
        // Try one more time
        sb.append(a);
        count++;
        
        if (sb.toString().contains(b)) {
            return count;
        }
        
        return -1;
    }
}

C++ Solution

class Solution {
public:
    int repeatedStringMatch(string a, string b) {
        string temp = a;
        int count = 1;
        
        // Calculate minimum repetitions needed
        while (temp.length() < b.length()) {
            temp += a;
            count++;
        }
        
        // Check if b is substring
        if (temp.find(b) != string::npos) {
            return count;
        }
        
        // Try one more time
        temp += a;
        if (temp.find(b) != string::npos) {
            return count + 1;
        }
        
        return -1;
    }
};

C# Solution

public class Solution {
    public int RepeatedStringMatch(string a, string b) {
        StringBuilder sb = new StringBuilder();
        int count = 0;
        
        // Add string a until length is at least equal to b
        while (sb.Length < b.Length) {
            sb.Append(a);
            count++;
        }
        
        // Check if b is substring
        if (sb.ToString().Contains(b)) {
            return count;
        }
        
        // Try one more time
        sb.Append(a);
        count++;
        
        if (sb.ToString().Contains(b)) {
            return count;
        }
        
        return -1;
    }
}

Implementation Notes:

  • Build string by repeating a until length is at least equal to b
  • Check if b is substring of the built string
  • If not found, try one more repetition
  • Time complexity: O(N), where N is length of final string
  • Space complexity: O(N)
