829. Consecutive Numbers Sum

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer n, return the number of ways to represent n as the sum of consecutive positive integers.

Examples:

Example 1:

Input: n = 5
Output: 2
Explanation: 5 = 2 + 3 = 5

Example 2:

Input: n = 9
Output: 3
Explanation: 9 = 4 + 5 = 2 + 3 + 4

Example 3:

Input: n = 15
Output: 4
Explanation: 15 = 8 + 7 = 4 + 5 + 6 = 1 + 2 + 3 + 4 + 5

Constraints:

  • 1 ≤ n ≤ 10^9

Python Solution

class Solution:
    def consecutiveNumbersSum(self, n: int) -> int:
        count = 1  # n itself is always a solution
        i = 2
        
        # For k consecutive numbers starting at x:
        # n = x + (x+1) + ... + (x+k-1)
        # n = kx + k(k-1)/2
        # x = (n - k(k-1)/2) / k must be positive integer
        while i * (i - 1) // 2 < n:
            if (n - i * (i - 1) // 2) % i == 0:
                count += 1
            i += 1
            
        return count

Implementation Notes:

  • Uses mathematical approach
  • Checks if n can be represented as sum of k consecutive numbers
  • Time complexity: O(√n)
  • Space complexity: O(1)

Java Solution

class Solution {
    public int consecutiveNumbersSum(int n) {
        int count = 1;  // n itself is always a solution
        int i = 2;
        
        while (i * (i - 1L) / 2 < n) {
            if ((n - i * (i - 1L) / 2) % i == 0) {
                count++;
            }
            i++;
        }
        
        return count;
    }
}
