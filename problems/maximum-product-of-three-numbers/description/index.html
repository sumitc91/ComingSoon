628. Maximum Product of Three Numbers

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer array nums, find three numbers whose product is maximum and return the maximum product.

Examples:

Example 1:

Input: nums = [1,2,3]
Output: 6
Explanation: The maximum product is achieved by multiplying 1 × 2 × 3 = 6.

Example 2:

Input: nums = [1,2,3,4]
Output: 24
Explanation: The maximum product is achieved by multiplying 2 × 3 × 4 = 24.

Example 3:

Input: nums = [-1,-2,-3]
Output: -6
Explanation: The maximum product is achieved by multiplying -1 × -2 × -3 = -6.

Constraints:

  • 3 ≤ nums.length ≤ 104
  • -1000 ≤ nums[i] ≤ 1000

Python Solution

class Solution:
    def maximumProduct(self, nums: List[int]) -> int:
        nums.sort()
        return max(nums[-1] * nums[-2] * nums[-3],
                  nums[0] * nums[1] * nums[-1])

Alternative Solution (O(n)):

class Solution:
    def maximumProduct(self, nums: List[int]) -> int:
        min1 = min2 = float('inf')
        max1 = max2 = max3 = float('-inf')
        
        for num in nums:
            if num <= min1:
                min2 = min1
                min1 = num
            elif num <= min2:
                min2 = num
                
            if num >= max1:
                max3 = max2
                max2 = max1
                max1 = num
            elif num >= max2:
                max3 = max2
                max2 = num
            elif num >= max3:
                max3 = num
                
        return max(min1 * min2 * max1, max1 * max2 * max3)

Implementation Notes:

  • First solution uses sorting: O(n log n) time, O(1) space
  • Second solution uses single pass: O(n) time, O(1) space
  • Handles both positive and negative numbers
  • Considers two cases: product of three largest numbers or product of two smallest (negative) numbers with largest number

class Solution {
    public int maximumProduct(int[] nums) {

        return Math.max(nums[n-1] * nums[n-2] * nums[n-3],
                       nums[0] * nums[1] * nums[n-1]);
    }
}
