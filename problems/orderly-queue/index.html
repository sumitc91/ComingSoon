899. Orderly Queue

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given a string s and an integer k. You can choose one of the first k letters of s and append it at the end of the string.

Return the lexicographically smallest string you could have after applying any number of moves.

Example 1:

Input: s = "cba", k = 1
Output: "acb"
Explanation: 
In the first move, we move the 1st character 'c' to the end, obtaining the string "bac".
In the second move, we move the 1st character 'b' to the end, obtaining the final result "acb".

Example 2:

Input: s = "baaca", k = 3
Output: "aaabc"
Explanation: 
In the first move, we move the 1st character 'b' to the end, obtaining the string "aacab".
In the second move, we move the 3rd character 'c' to the end, obtaining the final result "aaabc".

Constraints:

  • 1 <= k <= s.length <= 1000
  • s consist of lowercase English letters.

Solution

Python Solution

class Solution:
    def orderlyQueue(self, s: str, k: int) -> str:
        if k == 1:
            # For k=1, we can only rotate the string
            # Find the lexicographically smallest rotation
            return min(s[i:] + s[:i] for i in range(len(s)))
        else:
            # For k>1, we can get any permutation
            # Return the sorted string
            return ''.join(sorted(s))

Time Complexity: O(n²) for k=1, O(n log n) for k>1

For k=1, we need to check all possible rotations. For k>1, we just need to sort the string.

Space Complexity: O(n)

To store the rotated strings or sorted characters.

Java Solution

class Solution {
    public String orderlyQueue(String s, int k) {
        if (k == 1) {
            // For k=1, find the lexicographically smallest rotation
            String ans = s;
            for (int i = 0; i < s.length(); i++) {
                String temp = s.substring(i) + s.substring(0, i);
                if (temp.compareTo(ans) < 0) {
                    ans = temp;
                }
            }
            return ans;
        } else {
            // For k>1, return the sorted string
            char[] chars = s.toCharArray();
            Arrays.sort(chars);
            return new String(chars);
        }
    }
}

public class Solution {
    public string OrderlyQueue(string s, int k) {
        if (k == 1) {
            // For k=1, find the lexicographically smallest rotation
            string ans = s;
            for (int i = 0; i < s.Length; i++) {
                string temp = s.Substring(i) + s.Substring(0, i);
                if (string.Compare(temp, ans) < 0) {
                    ans = temp;
                }
            }
            return ans;
        } else {
            // For k>1, return the sorted string
            char[] chars = s.ToCharArray();
            Array.Sort(chars);
            return new string(chars);
        }
    }
}
