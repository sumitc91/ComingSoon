810. Chalkboard XOR Game

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an array of integers nums. Alice and Bob take turns playing a game, with Alice starting first.

During each player's turn, that player must pick an integer from nums and remove it. The player who removes a number that makes the bitwise XOR of all remaining elements equal to 0 loses. If a player cannot make a move, they lose.

Return true if Alice wins the game, assuming both players play optimally.

Examples:

Example 1:

Input: nums = [1,1,2]
Output: false
Explanation: 
Alice has two choices: erase 1 or erase 2. 
If she erases 1, the nums array becomes [1, 2]. The bitwise XOR of all remaining elements is 1 XOR 2 = 3. 
Now Bob can remove any element he wants, because Alice will be the one to remove the last element and she will lose. 
If Alice erases 2 first, now nums becomes [1, 1]. The bitwise XOR of all remaining elements is 1 XOR 1 = 0. 
Alice will lose.

Example 2:

Input: nums = [1,2,3]
Output: true

Example 3:

Input: nums = [1]
Output: true

Constraints:

  • 1 ≤ nums.length ≤ 1000
  • 0 ≤ nums[i] < 2¹⁶

Python Solution

class Solution:
    def xorGame(self, nums: List[int]) -> bool:
        if len(nums) % 2 == 0:
            return True
            
        xor_sum = 0
        for num in nums:
            xor_sum ^= num
            
        return xor_sum == 0

Implementation Notes:

  • If array length is even, Alice always wins
  • If array length is odd, Alice wins only if XOR of all numbers is 0
  • Time complexity: O(n)
  • Space complexity: O(1)

public class Solution {
    public bool XorGame(int[] nums) {
        if (nums.Length % 2 == 0) {
            return true;
        }
        
        int xorSum = 0;
        foreach (int num in nums) {
            xorSum ^= num;
        }
        
        return xorSum == 0;
    }
}

Implementation Notes:

  • Uses mathematical properties of XOR operation
  • For even length arrays, Alice can always win by maintaining XOR balance
  • For odd length arrays, Alice wins only if initial XOR is 0
