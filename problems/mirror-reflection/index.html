858. Mirror Reflection

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

There is a special square room with mirrors on each wall. Except for the southwest corner, there are receptors on each corner of the room. There is an emission device in the southwest corner.

The square room has walls of length p and a laser ray from the emission device bounces off the walls of the room and hits one of the receptors.

Given the two integers p and q, return the number of the receptor that the ray meets first. The test cases are guaranteed so that the ray will meet a receptor eventually.

Examples:

Example 1:

Input: p = 2, q = 1
Output: 2
Explanation: The ray meets receptor 2 after bouncing off the right wall.

Example 2:

Input: p = 3, q = 1
Output: 1

Constraints:

  • 1 ≤ q ≤ p ≤ 1000

Python Solution

class Solution:
    def mirrorReflection(self, p: int, q: int) -> int:
        # Reduce p and q to their smallest form
        while p % 2 == 0 and q % 2 == 0:
            p //= 2
            q //= 2
        
        # Check parity of p and q
        if p % 2 == 0:
            return 2  # Even p, odd q
        elif q % 2 == 0:
            return 0  # Odd p, even q
        else:
            return 1  # Both odd

Implementation Notes:

  • Uses mathematical properties of reflection
  • Time complexity: O(log min(p,q))
  • Space complexity: O(1)
