932. Beautiful Array

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

An array nums of length n is beautiful if:

  • nums is a permutation of the integers in the range [1, n].
  • For every 0 <= i < j < n, there is no index k with i < k < j where 2 * nums[k] == nums[i] + nums[j].

Given the integer n, return any beautiful array nums of length n. There will be at least one valid answer for the given n.

Example 1:

Input: n = 4
Output: [2,1,4,3]

Example 2:

Input: n = 5
Output: [3,1,2,5,4]

Constraints:

  • 1 <= n <= 1000

Solution

Python Solution

class Solution:
    def beautifulArray(self, n: int) -> List[int]:
        def helper(n):
            if n == 1:
                return [1]
            odds = helper((n + 1) // 2)
            evens = helper(n // 2)
            return [2 * x - 1 for x in odds] + [2 * x for x in evens]
        return helper(n)

Time Complexity: O(n)

We recursively build the array, visiting each number once.

Space Complexity: O(n)

We need space to store the result array.

Java Solution

class Solution {
    public int[] beautifulArray(int n) {
        List result = helper(n);
        return result.stream().mapToInt(i -> i).toArray();
    }
    
    private List helper(int n) {
        if (n == 1) {
            return Arrays.asList(1);
        }
        
        List odds = helper((n + 1) / 2);
        List evens = helper(n / 2);
        
        List result = new ArrayList<>();
        for (int x : odds) {
            result.add(2 * x - 1);
        }
        for (int x : evens) {
            result.add(2 * x);
        }
        return result;
    }
}

JavaScript Solution

/**
 * @param {number} n
 * @return {number[]}
 */
var beautifulArray = function(n) {
    function helper(n) {
        if (n === 1) {
            return [1];
        }
        
        const odds = helper(Math.floor((n + 1) / 2));
        const evens = helper(Math.floor(n / 2));
        
        return [...odds.map(x => 2 * x - 1), ...evens.map(x => 2 * x)];
    }
    
    return helper(n);
};
