893. Groups of Special-Equivalent Strings

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an array of strings of the same length words.

In one move, you can swap any two even indexed characters or any two odd indexed characters of a string.

Two strings are special-equivalent if they can be made equal after any number of moves.

Return the number of groups of special-equivalent strings.

Examples:

Example 1:

Input: words = ["abcd","cdab","cbad","xyzz","zzxy","zzyx"]
Output: 3
Explanation: 
One group is ["abcd","cdab","cbad"], since they are all pairwise special equivalent, and none of the other strings is special equivalent to these.
The other two groups are ["xyzz","zzxy"] and ["zzyx"].
Note that in particular, "zzxy" is not special equivalent to "zzyx".

Example 2:

Input: words = ["abc","acb","bac","bca","cab","cba"]
Output: 3

Constraints:

  • 1 ≤ words.length ≤ 1000
  • 1 ≤ words[i].length ≤ 20
  • All words[i] have the same length
  • words[i] consists of lowercase English letters

Python Solution

class Solution:
    def numSpecialEquivGroups(self, words: List[str]) -> int:
        def get_signature(word):
            # Separate even and odd indices
            even = sorted(word[0::2])
            odd = sorted(word[1::2])
            # Combine them into a signature
            return ''.join(even + odd)
        
        # Use set to count unique signatures
        return len(set(get_signature(word) for word in words))

Implementation Notes:

  • Creates a unique signature for each word
  • Time complexity: O(N * K * log K) where N is number of words and K is length of each word
  • Space complexity: O(N * K)

Java Solution

class Solution {
    public int numSpecialEquivGroups(String[] words) {
        Set seen = new HashSet<>();
        
        for (String word : words) {
            int[] count = new int[52];  // 26 for even indices, 26 for odd
            for (int i = 0; i < word.length(); ++i)
                count[word.charAt(i) - 'a' + 26 * (i % 2)]++;
            seen.add(Arrays.toString(count));
        }
        
        return seen.size();
    }
}

C++ Solution

class Solution {
public:
    int numSpecialEquivGroups(vector& words) {
        unordered_set seen;
        
        for (const string& word : words) {
            string even, odd;
            for (int i = 0; i < word.length(); i++) {
                if (i % 2 == 0)
                    even += word[i];
                else
                    odd += word[i];
            }
            sort(even.begin(), even.end());
            sort(odd.begin(), odd.end());
            seen.insert(even + odd);
        }
        
        return seen.size();
    }
};

JavaScript Solution

/**
 * @param {string[]} words
 * @return {number}
 */
var numSpecialEquivGroups = function(words) {
    const getSignature = (word) => {
        const even = [...word].filter((_, i) => i % 2 === 0).sort();
        const odd = [...word].filter((_, i) => i % 2 === 1).sort();
        return even.join('') + odd.join('');
    };
    
    return new Set(words.map(getSignature)).size;
};

C# Solution

public class Solution {
    public int NumSpecialEquivGroups(string[] words) {
        var seen = new HashSet();
        
        foreach (string word in words) {
            var count = new int[52];  // 26 for even indices, 26 for odd
            for (int i = 0; i < word.Length; i++)
                count[word[i] - 'a' + 26 * (i % 2)]++;
            seen.Add(string.Join(",", count));
        }
        
        return seen.Count;
    }
}

Implementation Notes:

  • Uses counting array for character frequencies
  • Separates even and odd indices
  • HashSet for unique group counting
