624. Maximum Distance in Arrays

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given m arrays, where each array is sorted in ascending order.

You can pick up two integers from two different arrays (each array picks one) and calculate the distance. We define the distance between two integers a and b to be their absolute difference |a - b|.

Return the maximum distance.

Examples:

Example 1:

Input: arrays = [[1,2,3],
         [4,5],
         [1,2,3]]
Output: 4
Explanation: One way to reach the maximum distance 4 is to pick 1 in the first or third array and pick 5 in the second array.

Example 2:

Input: arrays = [[1],
         [1]]
Output: 0

Constraints:

  • m == arrays.length
  • 2 <= m <= 10⁵
  • 1 <= arrays[i].length <= 500
  • -10⁴ <= arrays[i][j] <= 10⁴
  • arrays[i] is sorted in ascending order.
  • There will be at least two non-empty arrays.

Solution

Python Solution

class Solution:
    def maxDistance(self, arrays: List[List[int]]) -> int:
        result = 0
        min_val = arrays[0][0]
        max_val = arrays[0][-1]
        
        for i in range(1, len(arrays)):
            curr_min = arrays[i][0]
            curr_max = arrays[i][-1]
            
            # Try both combinations: current max - previous min and previous max - current min
            result = max(result, abs(curr_max - min_val), abs(max_val - curr_min))
            
            # Update min and max for next iteration
            min_val = min(min_val, curr_min)
            max_val = max(max_val, curr_max)
        
        return result

Time Complexity: O(n)

Where n is the number of arrays. We only need to look at the first and last elements of each array.

Space Complexity: O(1)

We only use a constant amount of extra space.

Explanation:

  1. Initialize min and max from first array
  2. For each subsequent array:
    • Calculate max distance using current array's min/max
    • Update overall min and max values
  3. Return the maximum distance found

Java Solution

class Solution {
    public int maxDistance(List> arrays) {
        int result = 0;
        int minVal = arrays.get(0).get(0);
        int maxVal = arrays.get(0).get(arrays.get(0).size() - 1);
        
        for (int i = 1; i < arrays.size(); i++) {
            List curr = arrays.get(i);
            int currMin = curr.get(0);
            int currMax = curr.get(curr.size() - 1);
            
            result = Math.max(result,
                     Math.max(Math.abs(currMax - minVal),
                             Math.abs(maxVal - currMin)));
            
            minVal = Math.min(minVal, currMin);
            maxVal = Math.max(maxVal, currMax);
        }
        
        return result;
    }
}

Alternative Solution using Two Passes:

class Solution {
public:
    int maxDistance(vector>& arrays) {
        // First pass: find global min and max
        int globalMin = INT_MAX;
        int globalMax = INT_MIN;
        
        for (const auto& arr : arrays) {
            globalMin = min(globalMin, arr[0]);
            globalMax = max(globalMax, arr.back());
        }
        
        // Second pass: find max distance excluding current array
        int result = 0;
        for (const auto& arr : arrays) {
            int currMin = arr[0];
            int currMax = arr.back();
            
            // If current array has the global min, use second min
            if (currMin == globalMin) {
                result = max(result, globalMax - arr[0]);
            }
            // If current array has the global max, use second max
            else if (currMax == globalMax) {
                result = max(result, arr.back() - globalMin);
            }
            // Otherwise, we can use both global min and max
            else {
                result = max(result, max(globalMax - currMin,
                                       currMax - globalMin));
            }
        }
        
        return result;
    }
};

JavaScript Solution

/**
 * @param {number[][]} arrays
 * @return {number}
 */
var maxDistance = function(arrays) {
    let result = 0;
    let minVal = arrays[0][0];
    let maxVal = arrays[0][arrays[0].length - 1];
    
    for (let i = 1; i < arrays.length; i++) {
        const currMin = arrays[i][0];
        const currMax = arrays[i][arrays[i].length - 1];
        
        result = Math.max(
            result,
            Math.abs(currMax - minVal),
            Math.abs(maxVal - currMin)
        );
        
        minVal = Math.min(minVal, currMin);
        maxVal = Math.max(maxVal, currMax);
    }
    
    return result;
};

Alternative Solution using Reduce:

var maxDistance = function(arrays) {
    const { min, max, result } = arrays.reduce(
        ({ min, max, result }, curr, i) => {
            if (i === 0) return { min: curr[0], max: curr[curr.length - 1], result: 0 };
            
            const newResult = Math.max(
                result,
                Math.abs(curr[curr.length - 1] - min),
                Math.abs(max - curr[0])
            );
            
            return {
                min: Math.min(min, curr[0]),
                max: Math.max(max, curr[curr.length - 1]),
                result: newResult
            };
        },
        { min: Infinity, max: -Infinity, result: 0 }
    );
    
    return result;
};

Implementation Notes:

  • The iterative solution is more readable and maintainable
  • The reduce solution is more functional but might be harder to understand
  • Both solutions have the same time and space complexity
  • The key is to maintain running min/max values while checking distances

C# Solution

public class Solution {
    public int MaxDistance(IList> arrays) {
        int result = 0;
        int minVal = arrays[0][0];
        int maxVal = arrays[0][arrays[0].Count - 1];
        
        for (int i = 1; i < arrays.Count; i++) {
            var curr = arrays[i];
            int currMin = curr[0];
            int currMax = curr[curr.Count - 1];
            
            result = Math.Max(result,
                     Math.Max(Math.Abs(currMax - minVal),
                             Math.Abs(maxVal - currMin)));
            
            minVal = Math.Min(minVal, currMin);
            maxVal = Math.Max(maxVal, currMax);
        }
        
        return result;
    }
}

Alternative Solution using LINQ:

public class Solution {
    public int MaxDistance(IList> arrays) {
        var result = arrays
            .Select((arr, i) => new { Min = arr.First(), Max = arr.Last(), Index = i })
            .Aggregate(
                new { MaxDist = 0, MinSoFar = arrays[0][0], MaxSoFar = arrays[0].Last() },
                (acc, curr) => {
                    if (curr.Index == 0) return acc;
                    
                    var maxDist = Math.Max(acc.MaxDist,
                                 Math.Max(Math.Abs(curr.Max - acc.MinSoFar),
                                        Math.Abs(acc.MaxSoFar - curr.Min)));
                    
                    return new {
                        MaxDist = maxDist,
                        MinSoFar = Math.Min(acc.MinSoFar, curr.Min),
                        MaxSoFar = Math.Max(acc.MaxSoFar, curr.Max)
                    };
                });
        
        return result.MaxDist;
    }
}

Implementation Notes:

  • First solution is more straightforward and easier to understand
  • LINQ solution is more functional but might be harder to read
  • Both solutions have O(n) time complexity
  • LINQ solution creates more temporary objects but is more declarative
