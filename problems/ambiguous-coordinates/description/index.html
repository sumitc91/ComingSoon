816. Ambiguous Coordinates

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

We had some 2-dimensional coordinates, like "(1, 3)" or "(2, 0.5)". Then, we removed all commas, decimal points, and spaces and ended up with the string s.

For example, "(1, 3)" becomes s = "(13)" and "(2, 0.5)" becomes s = "(205)".

Return a list of strings representing all possibilities for what our original coordinates could have been.

Our original representation never had extraneous zeroes, so we never started with numbers like "00", "0.0", "0.00", "1.0", "001", "00.01", or any other number that can be represented with fewer digits. Also, a decimal point within a number never occurs without at least one digit occurring before it, so we never started with numbers like ".1".

Examples:

Example 1:

Input: s = "(123)"
Output: ["(1, 2.3)","(1, 23)","(1.2, 3)","(12, 3)"]

Example 2:

Input: s = "(00011)"
Output: ["(0, 0.011)","(0.001, 1)"]

Example 3:

Input: s = "(0123)"
Output: ["(0, 1.23)","(0, 12.3)","(0, 123)","(0.1, 2.3)","(0.1, 23)","(0.12, 3)"]

Constraints:

  • 4 ≤ s.length ≤ 12
  • s[0] == '(' and s[s.length - 1] == ')'
  • The rest of s are digits

Python Solution

class Solution:
    def ambiguousCoordinates(self, s: str) -> List[str]:
        def make_nums(s: str) -> List[str]:
            if not s or len(s) > 1 and s[0] == '0' and s[-1] == '0':
                return []
            if len(s) > 1 and s[0] == '0':
                return ['0.' + s[1:]]
            if s[-1] == '0':
                return [s]
                
            nums = [s]
            for i in range(1, len(s)):
                nums.append(s[:i] + '.' + s[i:])
            return nums
            
        s = s[1:-1]  # Remove parentheses
        result = []
        
        for i in range(1, len(s)):
            left = make_nums(s[:i])
            right = make_nums(s[i:])
            
            for l in left:
                for r in right:
                    result.append(f"({l}, {r})")
                    
        return result

Implementation Notes:

  • Uses helper function to generate valid numbers from a string
  • Handles special cases with leading and trailing zeros
  • Time complexity: O(n³) where n is length of string
  • Space complexity: O(n³)

Java Solution

class Solution {
    public List ambiguousCoordinates(String s) {
        s = s.substring(1, s.length() - 1);
        List result = new ArrayList<>();
        
        for (int i = 1; i < s.length(); i++) {
            List left = makeNums(s.substring(0, i));
            List right = makeNums(s.substring(i));
            
            for (String l : left) {
                for (String r : right) {
                    result.add("(" + l + ", " + r + ")");
                }
            }
        }
        
        return result;
    }
    
    private List makeNums(String s) {
        List nums = new ArrayList<>();
        if (s.length() == 0 || (s.length() > 1 && s.charAt(0) == '0' && s.charAt(s.length() - 1) == '0')) {
            return nums;
        }
        if (s.length() > 1 && s.charAt(0) == '0') {
            nums.add("0." + s.substring(1));
            return nums;
        }
        if (s.charAt(s.length() - 1) == '0') {
            nums.add(s);
            return nums;
        }
        
        nums.add(s);
        for (int i = 1; i < s.length(); i++) {
            nums.add(s.substring(0, i) + "." + s.substring(i));
        }
        return nums;
    }
}

var ambiguousCoordinates = function(s) {
    s = s.slice(1, -1);
    const result = [];
    
    const makeNums = (s) => {
        if (!s || (s.length > 1 && s[0] === '0' && s[s.length - 1] === '0')) {
            return [];
        }
        if (s.length > 1 && s[0] === '0') {
            return ['0.' + s.slice(1)];
        }
        if (s[s.length - 1] === '0') {
            return [s];
        }
        
        const nums = [s];
        for (let i = 1; i < s.length; i++) {
            nums.push(s.slice(0, i) + '.' + s.slice(i));
        }
        return nums;
    };
    
    for (let i = 1; i < s.length; i++) {
        const left = makeNums(s.slice(0, i));
        const right = makeNums(s.slice(i));
        
        for (const l of left) {
            for (const r of right) {
                result.push(`(${l}, ${r})`);
            }
        }
    }
    
    return result;
};
