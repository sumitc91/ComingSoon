690. Employee Importance

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You have a data structure of employee information, including the employee's unique ID, importance value, and direct subordinates' IDs.

You are given an array of employees employees where:

  • employees[i].id is the ID of the ith employee
  • employees[i].importance is the importance value of the ith employee
  • employees[i].subordinates is a list of the IDs of the direct subordinates of the ith employee

Given an employee ID id, return the total importance value of this employee and all their direct and indirect subordinates.

Examples:

Example 1:

Input: employees = [[1,5,[2,3]],[2,3,[]],[3,3,[]]], id = 1
Output: 11
Explanation: Employee 1 has an importance value of 5 and has two direct subordinates: 2 and 3.
They both have an importance value of 3.
So the total importance value of employee 1 is 5 + 3 + 3 = 11.

Example 2:

Input: employees = [[1,2,[5]],[5,-3,[]]], id = 5
Output: -3

Constraints:

  • 1 ≤ employees.length ≤ 2000
  • 1 ≤ employees[i].id ≤ 2000
  • All employees[i].id are unique
  • -100 ≤ employees[i].importance ≤ 100
  • One employee has at most one direct leader and may have several subordinates
  • The IDs in employees[i].subordinates are valid IDs

Python Solution

"""
# Definition for Employee.
class Employee:
    def __init__(self, id: int, importance: int, subordinates: List[int]):
        self.id = id
        self.importance = importance
        self.subordinates = subordinates
"""

class Solution:
    def getImportance(self, employees: List['Employee'], id: int) -> int:
        # Create a dictionary to store employee objects by their ID
        emp_dict = {emp.id: emp for emp in employees}
        
        def dfs(emp_id: int) -> int:
            employee = emp_dict[emp_id]
            total = employee.importance
            
            # Add importance of all subordinates recursively
            for sub_id in employee.subordinates:
                total += dfs(sub_id)
                
            return total
        
        return dfs(id)

Implementation Notes:

  • Uses depth-first search (DFS) to traverse the employee hierarchy
  • Creates a dictionary for O(1) employee lookup by ID
  • Time complexity: O(N) where N is the total number of employees in the hierarchy
  • Space complexity: O(N) for the dictionary and recursion stack

Java Solution

/*
// Definition for Employee.
class Employee {
    public int id;
    public int importance;
    public List subordinates;

class Solution {
    Map employeeMap;
    
    public int getImportance(List employees, int id) {
        employeeMap = new HashMap<>();
        for (Employee emp : employees) {
            employeeMap.put(emp.id, emp);
        }
        
        return dfs(id);
    }
    
    private int dfs(int empId) {
        Employee employee = employeeMap.get(empId);
        int total = employee.importance;
        
        for (int subId : employee.subordinates) {
            total += dfs(subId);
        }
        
        return total;
    }
}

C# Solution

/*
// Definition for Employee.
class Employee {
    public int id;
    public int importance;
    public IList subordinates;
}
*/

class Solution {
    private Dictionary empDict;
    
    public int GetImportance(IList employees, int id) {
        empDict = new Dictionary();
        foreach (var emp in employees) {
            empDict[emp.id] = emp;
        }
        
        return DFS(id);
    }
    
    private int DFS(int empId) {
        Employee emp = empDict[empId];
        int total = emp.importance;
        
        foreach (int subId in emp.subordinates) {
            total += DFS(subId);
        }
        
        return total;
    }
}

Implementation Notes:

  • Uses depth-first search to calculate total importance
  • Stores employees in a dictionary/map for efficient lookup
  • Time complexity: O(N) where N is the number of employees
  • Space complexity: O(N) for the dictionary and recursion stack
