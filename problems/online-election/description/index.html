911. Online Election

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given two integer arrays persons and times. In an election, the ith vote was cast for persons[i] at time times[i].

For each query at a time t, find the person that was leading the election at time t. Votes cast at time t will count towards our query. In the case of a tie, the most recent vote (among tied candidates) wins.

Implement the TopVotedCandidate class:

  • TopVotedCandidate(int[] persons, int[] times) Initializes the object with the persons and times arrays.
  • int q(int t) Returns the number of the person that was leading the election at time t.

Example 1:

Input
["TopVotedCandidate", "q", "q", "q", "q", "q", "q"]
[[[0, 1, 1, 0, 0, 1, 0], [0, 5, 10, 15, 20, 25, 30]], [3], [12], [25], [15], [24], [8]]
Output
[null, 0, 1, 1, 0, 0, 1]

Explanation
At time 3, the votes are [0], and 0 is leading.
At time 12, the votes are [0,1,1], and 1 is leading.
At time 25, the votes are [0,1,1,0,0,1], and 1 is leading (as ties are won by most recent vote.)
At time 15, the votes are [0,1,1,0], and 0 is leading (as ties are won by most recent vote.)
At time 24, the votes are [0,1,1,0,0], and 0 is leading.
At time 8, the votes are [0,1,1], and 1 is leading.

Constraints:

  • 1 <= persons.length <= 5000
  • times.length == persons.length
  • 0 <= persons[i] < persons.length
  • 0 <= times[i] <= 10⁹
  • times is sorted in a strictly increasing order.
  • times[0] <= t <= 10⁹
  • At most 10⁴ calls will be made to q.

Solution

Python Solution

class TopVotedCandidate:
    def __init__(self, persons: List[int], times: List[int]):
        self.times = times
        self.leading = []  # Store the leading person at each time
        count = {}  # Count of votes for each person
        max_count = 0  # Maximum number of votes
        leader = -1  # Current leader
        
        for person in persons:
            count[person] = count.get(person, 0) + 1
            if count[person] >= max_count:
                max_count = count[person]
                leader = person
            self.leading.append(leader)
    
    def q(self, t: int) -> int:
        # Binary search to find the rightmost time <= t
        left, right = 0, len(self.times) - 1
        while left <= right:
            mid = (left + right) // 2
            if self.times[mid] <= t:
                left = mid + 1
            else:
                right = mid - 1
        return self.leading[right]

Time Complexity: O(n) for initialization, O(log n) for each query

Initialization takes O(n) time to process all votes. Each query uses binary search to find the appropriate time.

Space Complexity: O(n)

We store the leading person at each time point.

Java Solution

class TopVotedCandidate {
    private int[] times;
    private int[] leading;
    
    public TopVotedCandidate(int[] persons, int[] times) {
        this.times = times;
        this.leading = new int[persons.length];
        
        Map count = new HashMap<>();
        int maxCount = 0;
        int leader = -1;
        
        for (int i = 0; i < persons.length; i++) {
            int person = persons[i];
            count.put(person, count.getOrDefault(person, 0) + 1);
            
            if (count.get(person) >= maxCount) {
                maxCount = count.get(person);
                leader = person;
            }
            leading[i] = leader;
        }
    }
    
    public int q(int t) {
        // Binary search to find the rightmost time <= t
        int left = 0, right = times.length - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (times[mid] <= t) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return leading[right];
    }
}

C# Solution

public class TopVotedCandidate {
    private int[] times;
    private int[] leading;
    
    public TopVotedCandidate(int[] persons, int[] times) {
        this.times = times;
        this.leading = new int[persons.Length];
        
        var count = new Dictionary();
        int maxCount = 0;
        int leader = -1;
        
        for (int i = 0; i < persons.Length; i++) {
            int person = persons[i];
            if (!count.ContainsKey(person)) {
                count[person] = 0;
            }
            count[person]++;
            
            if (count[person] >= maxCount) {
                maxCount = count[person];
                leader = person;
            }
            leading[i] = leader;
        }
    }
    
    public int Q(int t) {
        // Binary search to find the rightmost time <= t
        int left = 0, right = times.Length - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (times[mid] <= t) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return leading[right];
    }
}
