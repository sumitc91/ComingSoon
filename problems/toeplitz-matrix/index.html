766. Toeplitz Matrix

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an m x n matrix, return true if the matrix is Toeplitz. Otherwise, return false.

A matrix is Toeplitz if every diagonal from top-left to bottom-right has the same elements.

Examples:

Example 1:

Input: matrix = [[1,2,3,4],[5,1,2,3],[9,5,1,2]]
Output: true
Explanation:
In the above grid, the diagonals are:
"[9]", "[5, 5]", "[1, 1, 1]", "[2, 2, 2]", "[3, 3]", "[4]".
In each diagonal all elements are the same, so the answer is True.

Example 2:

Input: matrix = [[1,2],[2,2]]
Output: false
Explanation:
The diagonal "[1, 2]" has different elements.

Constraints:

  • m == matrix.length
  • n == matrix[i].length
  • 1 ≤ m, n ≤ 20
  • 0 ≤ matrix[i][j] ≤ 99

Python Solution

class Solution:
    def isToeplitzMatrix(self, matrix: List[List[int]]) -> bool:
        rows, cols = len(matrix), len(matrix[0])
        
        # Check each diagonal starting from first row
        for j in range(cols):
            val = matrix[0][j]
            i, k = 1, j + 1
            while i < rows and k < cols:
                if matrix[i][k] != val:
                    return False
                i += 1
                k += 1
                
        # Check each diagonal starting from first column
        for i in range(1, rows):
            val = matrix[i][0]
            j, k = 1, i + 1
            while j < cols and k < rows:
                if matrix[k][j] != val:
                    return False
                j += 1
                k += 1
                
        return True

Implementation Notes:

  • Checks each diagonal starting from first row and first column
  • For each diagonal, verifies all elements are the same
  • Time complexity: O(m * n) where m and n are dimensions of matrix
  • Space complexity: O(1) as we only use a constant amount of extra space

class Solution {
    public boolean isToeplitzMatrix(int[][] matrix) {
        int rows = matrix.length;
        int cols = matrix[0].length;
        
        // Check each diagonal starting from first row
        for (int j = 0; j < cols; j++) {
            int val = matrix[0][j];
            int i = 1, k = j + 1;
            while (i < rows && k < cols) {
                if (matrix[i][k] != val) {
                    return false;
                }
                i++;
                k++;
            }
        }
        
        // Check each diagonal starting from first column
        for (int i = 1; i < rows; i++) {
            int val = matrix[i][0];
            int j = 1, k = i + 1;
            while (j < cols && k < rows) {
                if (matrix[k][j] != val) {
                    return false;
                }
                j++;
                k++;
            }
        }
        
        return true;
    }
}
