883. Projection Area of 3D Shapes

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an n x n grid where we place some 1 x 1 x 1 cubes that are axis-aligned with the x, y, and z axes.

Each value v = grid[i][j] represents a tower of v cubes placed on top of the cell (i, j).

We view the projection of these cubes onto the xy, yz, and zx planes. A projection is like a shadow, that maps our 3D figure to a 2D plane. We are viewing the "shadow" when looking at the cubes from the top, front, and side.

Return the total area of all three projections.

Examples:

Example 1:

Input: grid = [[1,2],[3,4]]
Output: 17
Explanation: Here are the three projections ("shadows") of the shape made with each axis-aligned plane.

Example 2:

Input: grid = [[2]]
Output: 5

Example 3:

Input: grid = [[1,0],[0,2]]
Output: 8

Constraints:

  • n == grid.length == grid[i].length
  • 1 ≤ n ≤ 50
  • 0 ≤ grid[i][j] ≤ 50

Python Solution

class Solution:
    def projectionArea(self, grid: List[List[int]]) -> int:
        n = len(grid)
        top = sum(cell > 0 for row in grid for cell in row)
        front = sum(max(row) for row in grid)
        side = sum(max(grid[i][j] for i in range(n)) for j in range(n))
        return top + front + side

Implementation Notes:

  • Top view: Count non-zero cells
  • Front view: Maximum in each row
  • Side view: Maximum in each column
  • Time complexity: O(n²)
  • Space complexity: O(1)

Java Solution

class Solution {
    public int projectionArea(int[][] grid) {
        int n = grid.length;
        int top = 0, front = 0, side = 0;
        
        for (int i = 0; i < n; i++) {
            int maxRow = 0, maxCol = 0;
            for (int j = 0; j < n; j++) {
                if (grid[i][j] > 0) top++;
                maxRow = Math.max(maxRow, grid[i][j]);
                maxCol = Math.max(maxCol, grid[j][i]);
            }
            front += maxRow;
            side += maxCol;
        }
        
        return top + front + side;
    }
}
