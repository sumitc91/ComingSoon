960. Delete Columns to Make Sorted III

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an array of n strings strs, all of the same length. We may choose any deletion indices, and we delete all the characters in those indices for each string.

For example, if we have strs = ["abcdef","uvwxyz"] and deletion indices {0, 2, 3}, then the final array after deletions is ["bef", "vyz"].

Suppose we chose a set of deletion indices answer such that after deletions, the final array has its elements in lexicographic order (i.e., strs[0] ≤ strs[1] ≤ strs[2] ≤ ... ≤ strs[n - 1]). Return the minimum possible value of answer.length.

Example 1:

Input: strs = ["babca","bbazb"]
Output: 3
Explanation: After deleting columns 0, 1, and 4, the final array is strs = ["bc", "az"].
Both these rows are individually in lexicographic order (ie. strs[0][0] <= strs[0][1] and strs[1][0] <= strs[1][1]).
Note that strs[0] > strs[1] - the array's rows are not necessarily in lexicographic order.

Example 2:

Input: strs = ["edcba"]
Output: 4
Explanation: If we delete less than 4 columns, the only resulting row is not in lexicographic order.

Example 3:

Input: strs = ["ghi","def","abc"]
Output: 0
Explanation: All rows are already lexicographically sorted.

Constraints:

  • n == strs.length
  • 1 <= n <= 100
  • 1 <= strs[i].length <= 100
  • strs[i] consists of lowercase English letters.

Solution

Python Solution

class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        n, m = len(strs), len(strs[0])
        # dp[i] represents the length of the longest non-decreasing subsequence ending at index i
        dp = [1] * m
        
        for i in range(1, m):
            for j in range(i):
                # Check if we can extend the subsequence ending at j with i
                if all(strs[k][j] <= strs[k][i] for k in range(n)):
                    dp[i] = max(dp[i], dp[j] + 1)
        
        return m - max(dp)

Time Complexity: O(m² * n)

Where m is the length of each string and n is the number of strings. We need to check each pair of columns and each string.

Space Complexity: O(m)

We need to store the dp array of length m.

Java Solution

class Solution {
    public int minDeletionSize(String[] strs) {
        int n = strs.length, m = strs[0].length();
        // dp[i] represents the length of the longest non-decreasing subsequence ending at index i
        int[] dp = new int[m];
        Arrays.fill(dp, 1);
        
        for (int i = 1; i < m; i++) {
            for (int j = 0; j < i; j++) {
                // Check if we can extend the subsequence ending at j with i
                boolean canExtend = true;
                for (int k = 0; k < n; k++) {
                    if (strs[k].charAt(j) > strs[k].charAt(i)) {
                        canExtend = false;
                        break;
                    }
                }
                if (canExtend) {
                    dp[i] = Math.max(dp[i], dp[j] + 1);
                }
            }
        }
        
        int maxLen = 0;
        for (int len : dp) {
            maxLen = Math.max(maxLen, len);
        }
        return m - maxLen;
    }
}
