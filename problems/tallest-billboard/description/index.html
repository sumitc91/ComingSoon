956. Tallest Billboard

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are installing a billboard and want it to have the largest height. The billboard will have two steel supports, one on each side. Each steel support must be an equal height.

You are given a collection of rods that can be welded together. For example, if you have rods of lengths 1, 2, and 3, you can weld them together to make a support of length 6.

Return the largest possible height of your billboard installation. If you cannot support the billboard, return 0.

Example 1:

Input: rods = [1,2,3,6]
Output: 6
Explanation: We have two disjoint subsets {1,2,3} and {6}, which have the same sum = 6.

Example 2:

Input: rods = [1,2,3,4,5,6]
Output: 10
Explanation: We have two disjoint subsets {2,3,5} and {4,6}, which have the same sum = 10.

Example 3:

Input: rods = [1,2]
Output: 0
Explanation: The billboard cannot be supported, so we return 0.

Constraints:

  • 1 <= rods.length <= 20
  • 1 <= rods[i] <= 1000
  • sum(rods[i]) <= 5000

Solution

Python Solution

class Solution:
    def tallestBillboard(self, rods: List[int]) -> int:
        # dp[diff] represents the maximum sum of the shorter support when the difference is diff
        dp = {0: 0}
        
        for rod in rods:
            new_dp = {}
            for diff, shorter in dp.items():
                # Add rod to shorter support
                new_diff = diff + rod
                new_shorter = shorter
                if new_diff not in new_dp or new_shorter > new_dp[new_diff]:
                    new_dp[new_diff] = new_shorter
                
                # Add rod to longer support
                new_diff = abs(diff - rod)
                new_shorter = max(shorter, shorter + diff - rod)
                if new_diff not in new_dp or new_shorter > new_dp[new_diff]:
                    new_dp[new_diff] = new_shorter
                
                # Keep the current state
                if diff not in new_dp or shorter > new_dp[diff]:
                    new_dp[diff] = shorter
            
            dp = new_dp
        
        return dp[0]

Time Complexity: O(n * sum)

Where n is the number of rods and sum is the sum of all rod lengths. We need to process each rod and each possible difference.

Space Complexity: O(sum)

We need to store the states for each possible difference.

Java Solution

class Solution {
    public int tallestBillboard(int[] rods) {
        // dp[diff] represents the maximum sum of the shorter support when the difference is diff
        Map dp = new HashMap<>();
        dp.put(0, 0);
        
        for (int rod : rods) {
            Map newDp = new HashMap<>();
            for (Map.Entry entry : dp.entrySet()) {
                int diff = entry.getKey();
                int shorter = entry.getValue();
                
                // Add rod to shorter support
                int newDiff = diff + rod;
                int newShorter = shorter;
                newDp.put(newDiff, Math.max(newDp.getOrDefault(newDiff, 0), newShorter));
                
                // Add rod to longer support
                newDiff = Math.abs(diff - rod);
                newShorter = Math.max(shorter, shorter + diff - rod);
                newDp.put(newDiff, Math.max(newDp.getOrDefault(newDiff, 0), newShorter));
                
                // Keep the current state
                newDp.put(diff, Math.max(newDp.getOrDefault(diff, 0), shorter));
            }
            dp = newDp;
        }
        
        return dp.get(0);
    }
}

C# Solution

public class Solution {
    public int TallestBillboard(int[] rods) {
        // dp[diff] represents the maximum sum of the shorter support when the difference is diff
        Dictionary dp = new Dictionary();
        dp[0] = 0;
        
        foreach (int rod in rods) {
            Dictionary newDp = new Dictionary();
            foreach (var kvp in dp) {
                int diff = kvp.Key;
                int shorter = kvp.Value;
                
                // Add rod to shorter support
                int newDiff = diff + rod;
                int newShorter = shorter;
                if (!newDp.ContainsKey(newDiff) || newShorter > newDp[newDiff]) {
                    newDp[newDiff] = newShorter;
                }
                
                // Add rod to longer support
                newDiff = Math.Abs(diff - rod);
                newShorter = Math.Max(shorter, shorter + diff - rod);
                if (!newDp.ContainsKey(newDiff) || newShorter > newDp[newDiff]) {
                    newDp[newDiff] = newShorter;
                }
                
                // Keep the current state
                if (!newDp.ContainsKey(diff) || shorter > newDp[diff]) {
                    newDp[diff] = shorter;
                }
            }
            dp = newDp;
        }
        
        return dp[0];
    }
}
