756. Pyramid Transition Matrix

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

We are stacking blocks to form a pyramid. Each block has a color which is a one letter string, like 'Z'.

For every block of color 'C' we place not in the bottom row, we are placing it on top of a left block of color 'A' and right block of color 'B'. We are allowed to place the block there only if '(A, B, C)' is an allowed triple.

We start with a bottom row of bottom, represented as a single string. We also start with a list of allowed triples allowed. Each allowed triple is represented as a string of length 3.

Return true if we can build the pyramid all the way to the top, otherwise false.

Examples:

Example 1:

Input: bottom = "BCD", allowed = ["BCG", "CDE", "GEA", "FFF"]
Output: true
Explanation:
We can stack the pyramid like this:
    A
   / \
  G   E
 / \ / \
B   C   D

We are allowed to place G on top of B and C because BCG is an allowed triple.  Similarly, we can place E on top of C and D, then A on top of G and E.

Example 2:

Input: bottom = "AABA", allowed = ["AAA", "AAB", "ABA", "ABB", "BAC"]
Output: false
Explanation:
We can't stack the pyramid to the top. Note that there could be multiple triples that would be allowed in this case.

Constraints:

  • 2 ≤ bottom.length ≤ 8
  • 0 ≤ allowed.length ≤ 200
  • allowed[i].length == 3
  • The letters in all input strings are from the set {'A', 'B', 'C', 'D', 'E', 'F', 'G'}

Python Solution

class Solution:
    def pyramidTransition(self, bottom: str, allowed: List[str]) -> bool:
        # Build transition map
        transitions = {}
        for triple in allowed:
            key = (triple[0], triple[1])
            if key not in transitions:
                transitions[key] = set()
            transitions[key].add(triple[2])
            
        def can_build(row: str) -> bool:
            if len(row) == 1:
                return True
                
            # Try all possible next rows
            next_rows = []
            for i in range(len(row) - 1):
                key = (row[i], row[i + 1])
                if key not in transitions:
                    return False
                if not next_rows:
                    next_rows = [c for c in transitions[key]]
                else:
                    next_rows = [r + c for r in next_rows for c in transitions[key]]
                    
            # Try each possible next row
            for next_row in next_rows:
                if can_build(next_row):
                    return True
            return False
            
        return can_build(bottom)

Implementation Notes:

  • Uses DFS to try all possible pyramid configurations
  • Builds a transition map for O(1) lookup of allowed triples
  • Time complexity: O(7^n) where n is the length of bottom
  • Space complexity: O(7^n) for storing possible next rows

Java Solution

class Solution {
    private Map> transitions;
    
    public boolean pyramidTransition(String bottom, List allowed) {
        // Build transition map
        transitions = new HashMap<>();
        for (String triple : allowed) {
            String key = triple.substring(0, 2);
            transitions.computeIfAbsent(key, k -> new HashSet<>())
                     .add(triple.charAt(2));
        }
        
        return canBuild(bottom);
    }
    
    private boolean canBuild(String row) {
        if (row.length() == 1) {
            return true;
        }
        
        // Try all possible next rows
        List nextRows = new ArrayList<>();
        for (int i = 0; i < row.length() - 1; i++) {
            String key = row.substring(i, i + 2);
            if (!transitions.containsKey(key)) {
                return false;
            }
            
            if (nextRows.isEmpty()) {
                for (char c : transitions.get(key)) {
                    nextRows.add(String.valueOf(c));
                }
            } else {
                List newRows = new ArrayList<>();
                for (String prev : nextRows) {
                    for (char c : transitions.get(key)) {
                        newRows.add(prev + c);
                    }
                }
                nextRows = newRows;
            }
        }
        
        // Try each possible next row
        for (String nextRow : nextRows) {
            if (canBuild(nextRow)) {
                return true;
            }
        }
        
        return false;
    }
}

JavaScript Solution

function pyramidTransition(bottom, allowed) {
    // Build transition map
    const transitions = new Map();
    for (const triple of allowed) {
        const key = triple.slice(0, 2);
        if (!transitions.has(key)) {
            transitions.set(key, new Set());
        }
        transitions.get(key).add(triple[2]);
    }
    
    function canBuild(row) {
        if (row.length === 1) {
            return true;
        }
        
        // Try all possible next rows
        let nextRows = [];
        for (let i = 0; i < row.length - 1; i++) {
            const key = row.slice(i, i + 2);
            if (!transitions.has(key)) {
                return false;
            }
            
            if (nextRows.length === 0) {
                nextRows = Array.from(transitions.get(key)).map(c => c);
            } else {
                const newRows = [];
                for (const prev of nextRows) {
                    for (const c of transitions.get(key)) {
                        newRows.push(prev + c);
                    }
                }
                nextRows = newRows;
            }
        }
        
        // Try each possible next row
        for (const nextRow of nextRows) {
            if (canBuild(nextRow)) {
                return true;
            }
        }
        
        return false;
    }
    
    return canBuild(bottom);
}

C# Solution

public class Solution {
    private Dictionary> transitions;
    
    public bool PyramidTransition(string bottom, IList allowed) {
        // Build transition map
        transitions = new Dictionary>();
        foreach (string triple in allowed) {
            string key = triple.Substring(0, 2);
            if (!transitions.ContainsKey(key)) {
                transitions[key] = new HashSet();
            }
            transitions[key].Add(triple[2]);
        }
        
        return CanBuild(bottom);
    }
    
    private bool CanBuild(string row) {
        if (row.Length == 1) {
            return true;
        }
        
        // Try all possible next rows
        List nextRows = new List();
        for (int i = 0; i < row.Length - 1; i++) {
            string key = row.Substring(i, 2);
            if (!transitions.ContainsKey(key)) {
                return false;
            }
            
            if (nextRows.Count == 0) {
                nextRows = transitions[key].Select(c => c.ToString()).ToList();
            } else {
                List newRows = new List();
                foreach (string prev in nextRows) {
                    foreach (char c in transitions[key]) {
                        newRows.Add(prev + c);
                    }
                }
                nextRows = newRows;
            }
        }
        
        // Try each possible next row
        foreach (string nextRow in nextRows) {
            if (CanBuild(nextRow)) {
                return true;
            }
        }
        
        return false;
    }
}

Implementation Notes:

  • Uses DFS to try all possible pyramid configurations
  • Builds a transition map for O(1) lookup of allowed triples
  • Time complexity: O(7^n) where n is the length of bottom
  • Space complexity: O(7^n) for storing possible next rows
