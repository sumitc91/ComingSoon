957. Prison Cells After N Days

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

There are 8 prison cells in a row, and each cell is either occupied or vacant.

Each day, whether the cell is occupied or vacant changes according to the following rules:

  • If a cell has two adjacent neighbors that are both occupied or both vacant, then the cell becomes occupied.
  • Otherwise, it becomes vacant.

(Note that because the prison is a row, the first and the last cells in the row can't have two adjacent neighbors.)

We describe the current state of the prison in the following way: cells[i] == 1 if the i-th cell is occupied, else cells[i] == 0.

Given the initial state of the prison, return the state of the prison after N days (and N such changes described above.)

Example 1:

Input: cells = [0,1,0,1,1,0,0,1], N = 7
Output: [0,0,1,1,0,0,0,0]
Explanation: 
The following table summarizes the state of the prison on each day:
Day 0: [0, 1, 0, 1, 1, 0, 0, 1]
Day 1: [0, 1, 1, 0, 0, 0, 0, 0]
Day 2: [0, 0, 0, 0, 1, 1, 1, 0]
Day 3: [0, 1, 1, 0, 0, 1, 0, 0]
Day 4: [0, 0, 0, 0, 0, 1, 0, 0]
Day 5: [0, 1, 1, 1, 0, 1, 0, 0]
Day 6: [0, 0, 1, 0, 1, 1, 0, 0]
Day 7: [0, 0, 1, 1, 0, 0, 0, 0]

Example 2:

Input: cells = [1,0,0,1,0,0,1,0], N = 1000000000
Output: [0,0,1,1,1,1,1,0]

Constraints:

  • cells.length == 8
  • cells[i] is in {0, 1}
  • 1 <= N <= 10^9

Solution

Python Solution

class Solution:
    def prisonAfterNDays(self, cells: List[int], N: int) -> List[int]:
        # Convert cells to tuple for hashing
        state = tuple(cells)
        seen = {}
        
        for day in range(N):
            if state in seen:
                # We found a cycle, calculate remaining days
                cycle_length = day - seen[state]
                remaining_days = (N - day) % cycle_length
                # Get the state after remaining days
                for _ in range(remaining_days):
                    state = self.next_state(state)
                return list(state)
            
            seen[state] = day
            state = self.next_state(state)
        
        return list(state)
    
    def next_state(self, state: tuple) -> tuple:
        new_state = [0] * 8
        for i in range(1, 7):
            if state[i-1] == state[i+1]:
                new_state[i] = 1
        return tuple(new_state)

Time Complexity: O(1)

Since there are only 2^8 = 256 possible states, we will find a cycle within 256 days.

Space Complexity: O(1)

We store at most 256 states in our hash map.

class Solution {
    public int[] prisonAfterNDays(int[] cells, int N) {
        // Convert cells to string for hashing
        String state = Arrays.toString(cells);
        Map seen = new HashMap<>();
        
        for (int day = 0; day < N; day++) {
            if (seen.containsKey(state)) {
                // We found a cycle, calculate remaining days
                int cycleLength = day - seen.get(state);
                int remainingDays = (N - day) % cycleLength;
                // Get the state after remaining days
                for (int i = 0; i < remainingDays; i++) {
                    cells = nextState(cells);
                }
                return cells;
            }
            
            seen.put(state, day);
            cells = nextState(cells);
            state = Arrays.toString(cells);
        }
        
        return cells;
    }

            if (cells[i-1] == cells[i+1]) {
                newState[i] = 1;
            }
        }
        return newState;
    }
}
