693. Binary Number with Alternating Bits

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a positive integer n, return true if and only if it has alternating bits: namely, if two adjacent bits will always have different values.

Examples:

Example 1:

Input: n = 5
Output: true
Explanation: The binary representation of 5 is: 101

Example 2:

Input: n = 7
Output: false
Explanation: The binary representation of 7 is: 111

Example 3:

Input: n = 11
Output: false
Explanation: The binary representation of 11 is: 1011

Constraints:

  • 1 ≤ n ≤ 231 - 1

Python Solution

class Solution:
    def hasAlternatingBits(self, n: int) -> bool:
        # Right shift by 1 and XOR with original number
        # If alternating bits, all bits in result will be 1
        xor = n ^ (n >> 1)
        
        # Check if xor is all 1s
        # If all 1s, adding 1 will make it power of 2
        return (xor & (xor + 1)) == 0

Implementation Notes:

  • Uses bit manipulation to check alternating bits
  • XOR with right shift creates all 1s if alternating
  • Time complexity: O(1)
  • Space complexity: O(1)
