793. Preimage Size of Factorial Zeroes Function

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Let f(x) be the number of zeroes at the end of x!. Recall that x! = 1 * 2 * 3 * ... * x and by convention, 0! = 1.

For example, f(3) = 0 because 3! = 6 has no zeroes at the end, while f(11) = 2 because 11! = 39916800 has 2 zeroes at the end.

Given an integer k, return the number of non-negative integers x have the property that f(x) = k.

Examples:

Example 1:

Input: k = 0
Output: 5
Explanation: 0!, 1!, 2!, 3!, and 4! end with k = 0 zeroes.

Example 2:

Input: k = 5
Output: 0
Explanation: There is no x such that x! ends in k = 5 zeroes.

Example 3:

Input: k = 3
Output: 5

Constraints:

  • 0 ≤ k ≤ 10^9

Python Solution

class Solution:
    def preimageSizeFZF(self, k: int) -> int:
        def trailingZeroes(n: int) -> int:
            count = 0
            while n:
                n //= 5
                count += n
            return count
            
        def search(k: int) -> int:
            left, right = 0, 5 * (k + 1)
            while left <= right:
                mid = (left + right) // 2
                zeros = trailingZeroes(mid)
                if zeros == k:
                    return mid
                elif zeros < k:
                    left = mid + 1
                else:
                    right = mid - 1
            return -1
            
        # Find the first number with k trailing zeros
        first = search(k)
        if first == -1:
            return 0
            
        # Find the first number with k+1 trailing zeros
        last = search(k + 1)
        if last == -1:
            last = 5 * (k + 1)
            
        return last - first

Implementation Notes:

  • Uses binary search to find numbers with k trailing zeros
  • Counts trailing zeros by counting factors of 5
  • Returns difference between first number with k+1 zeros and first number with k zeros
  • Time complexity: O(log k)
  • Space complexity: O(1)

Java Solution

class Solution {
    public int preimageSizeFZF(int k) {
        long left = search(k);
        if (left == -1) return 0;
        long right = search(k + 1);
        if (right == -1) right = 5L * (k + 1);
        return (int)(right - left);
    }
    
    private long search(int k) {
        long left = 0, right = 5L * (k + 1);
        while (left <= right) {
            long mid = left + (right - left) / 2;
            long zeros = trailingZeroes(mid);
            if (zeros == k) return mid;
            else if (zeros < k) left = mid + 1;
            else right = mid - 1;
        }
        return -1;
    }
    
    private long trailingZeroes(long n) {
        long count = 0;
        while (n > 0) {
            n /= 5;
            count += n;
        }
        return count;
    }
}

JavaScript Solution

function preimageSizeFZF(k) {
    function trailingZeroes(n) {
        let count = 0;
        while (n > 0) {
            n = Math.floor(n / 5);
            count += n;
        }
        return count;
    }
    
    function search(k) {
        let left = 0, right = 5 * (k + 1);
        while (left <= right) {
            const mid = Math.floor(left + (right - left) / 2);
            const zeros = trailingZeroes(mid);
            if (zeros === k) return mid;
            else if (zeros < k) left = mid + 1;
            else right = mid - 1;
        }
        return -1;
    }
    
    const left = search(k);
    if (left === -1) return 0;
    const right = search(k + 1);
    if (right === -1) return 5 * (k + 1) - left;
    return right - left;
}
