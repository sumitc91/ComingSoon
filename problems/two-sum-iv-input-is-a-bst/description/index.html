653. Two Sum IV - Input is a BST

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given the root of a Binary Search Tree and a target number k, return true if there exist two elements in the BST such that their sum is equal to the given target.

Examples:

Example 1:

Input: root = [5,3,6,2,4,null,7], k = 9
Output: true
Explanation: The BST contains nodes with values 2, 3, 4, 5, 6, 7.
There exists two nodes with values 4 and 5 whose sum equals 9.

Example 2:

Input: root = [5,3,6,2,4,null,7], k = 28
Output: false
Explanation: No two nodes in the tree have a sum equal to 28.

Constraints:

  • The number of nodes in the tree is in the range [1, 104]
  • -104 ≤ Node.val ≤ 104
  • root is guaranteed to be a valid binary search tree
  • -105 ≤ k ≤ 105

Python Solution

class Solution:
    def findTarget(self, root: Optional[TreeNode], k: int) -> bool:
        def inorder(node, seen):
            if not node:
                return False
            
            # Check if we've seen k - current value
            if k - node.val in seen:
                return True
            
            # Add current value to seen set
            seen.add(node.val)
            
            # Check left and right subtrees
            return inorder(node.left, seen) or inorder(node.right, seen)
        
        return inorder(root, set())

Alternative Solution (Using Two Pointers with BST Iterator):

class BSTIterator:
    def __init__(self, root: TreeNode, forward: bool):
        self.stack = []
        self.forward = forward
        node = root
        while node:
            self.stack.append(node)
            node = node.right if forward else node.left
    
    def next(self) -> int:
        node = self.stack.pop()
        curr = node.left if self.forward else node.right
        while curr:
            self.stack.append(curr)
            curr = curr.right if self.forward else curr.left
        return node.val

class Solution:
    def findTarget(self, root: Optional[TreeNode], k: int) -> bool:
        if not root:
            return False
            
        left = BSTIterator(root, False)  # Forward iterator
        right = BSTIterator(root, True)  # Reverse iterator
        
        l, r = left.next(), right.next()
        while l < r:
            curr_sum = l + r
            if curr_sum == k:
                return True
            elif curr_sum < k:
                l = left.next()
            else:
                r = right.next()
        
        return False

Implementation Notes:

  • First solution uses HashSet with O(n) space
  • Second solution uses BST property with O(h) space
  • Both solutions have O(n) time complexity

Alternative Solution (Using LINQ):

public class Solution {
    public bool FindTarget(TreeNode root, int k) {
        var values = new HashSet();
        
        return TraverseTree(root)
               .Any(val => values.Contains(k - val) || !values.Add(val));
    }
    
    private IEnumerable TraverseTree(TreeNode node) {
        if (node == null) yield break;
        
        foreach (var val in TraverseTree(node.left))
            yield return val;
            
        yield return node.val;
        
        foreach (var val in TraverseTree(node.right))
            yield return val;
    }
}
