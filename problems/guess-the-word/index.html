843. Guess the Word

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an array of unique strings wordlist where wordlist[i] is 6 letters long, and one word in this list is chosen as secret.

You may call Master.guess(word) to guess a word. The guessed word should have type string and must be from the original list with 6 lowercase letters.

This function returns an integer type, representing the number of exact matches (value and position) of your guess to the secret word. Also, if your guess is not in the given wordlist, it will return -1 instead.

For each test case, you have exactly 10 guesses to guess the word. At the end of any number of calls, if you have made 10 or more calls to Master.guess and at least one of these guesses was secret, you pass the test case.

Examples:

Example 1:

Input: secret = "acckzz", wordlist = ["acckzz","ccbazz","eiowzz","abcczz"]
Output: You made the correct guess at some point.
Explanation:
master.guess("aaaaaa") returns -1, because "aaaaaa" is not in wordlist.
master.guess("acckzz") returns 6, because "acckzz" is secret and has all 6 matches.
master.guess("ccbazz") returns 3, because "ccbazz" has 3 matches.
master.guess("eiowzz") returns 2, because "eiowzz" has 2 matches.
master.guess("abcczz") returns 4, because "abcczz" has 4 matches.

Example 2:

Input: secret = "hamada", wordlist = ["hamada","khaled"]
Output: You guessed the secret word correctly.

Constraints:

  • 1 ≤ wordlist.length ≤ 100
  • wordlist[i].length == 6
  • wordlist[i] consists of lowercase English letters
  • All words in wordlist are unique
  • secret exists in wordlist
  • At most 10 calls will be made to Master.guess

Python Solution

class Solution:
    def findSecretWord(self, wordlist: List[str], master: 'Master') -> None:
        def matches(word1: str, word2: str) -> int:
            return sum(c1 == c2 for c1, c2 in zip(word1, word2))
        
        def getMostCommonWord(candidates: List[str], possible: List[str]) -> str:
            if not candidates:
                return possible[0]
            
            counts = [[0] * 26 for _ in range(6)]
            for word in possible:
                for i, c in enumerate(word):
                    counts[i][ord(c) - ord('a')] += 1
            
            maxScore = 0
            bestWord = candidates[0]
            
            for word in candidates:
                score = 0
                for i, c in enumerate(word):
                    score += counts[i][ord(c) - ord('a')]
                if score > maxScore:
                    maxScore = score
                    bestWord = word
            
            return bestWord
        
        n = len(wordlist)
        possible = wordlist[:]
        
        for _ in range(10):
            guess = getMostCommonWord(possible[:], possible)
            matches_count = master.guess(guess)
            
            if matches_count == 6:
                return
            
            possible = [w for w in possible if matches(w, guess) == matches_count]

Implementation Notes:

  • Uses minimax strategy to minimize worst-case scenarios
  • Maintains a list of possible words based on match counts
  • Time complexity: O(N²) where N is the length of wordlist
  • Space complexity: O(N)

Java Solution

class Solution {
    public void findSecretWord(String[] wordlist, Master master) {
        List possible = new ArrayList<>(Arrays.asList(wordlist));
        
        for (int i = 0; i < 10 && !possible.isEmpty(); i++) {
            String guess = getMostCommonWord(possible, possible);
            int matches = master.guess(guess);
            
            if (matches == 6) return;
            
            List newPossible = new ArrayList<>();
            for (String word : possible) {
                if (matches(word, guess) == matches) {
                    newPossible.add(word);
                }
            }
            possible = newPossible;
        }
    }
    
    private int matches(String word1, String word2) {
        int matches = 0;
        for (int i = 0; i < 6; i++) {
            if (word1.charAt(i) == word2.charAt(i)) {
                matches++;
            }
        }
        return matches;
    }
    
    private String getMostCommonWord(List candidates, List possible) {
        if (candidates.isEmpty()) {
            return possible.get(0);
        }
        
        int[][] counts = new int[6][26];
        for (String word : possible) {
            for (int i = 0; i < 6; i++) {
                counts[i][word.charAt(i) - 'a']++;
            }
        }
        
        int maxScore = 0;
        String bestWord = candidates.get(0);
        
        for (String word : candidates) {
            int score = 0;
            for (int i = 0; i < 6; i++) {
                score += counts[i][word.charAt(i) - 'a'];
            }
            if (score > maxScore) {
                maxScore = score;
                bestWord = word;
            }
        }
        
        return bestWord;
    }
}

/**
 * @param {string[]} wordlist
 * @param {Master} master
 * @return {void}

var findSecretWord = function(wordlist, master) {
    const matches = (word1, word2) => {
        let count = 0;
        for (let i = 0; i < 6; i++) {
            if (word1[i] === word2[i]) count++;
        }
        return count;
    };
    
    const getMostCommonWord = (candidates, possible) => {
        if (!candidates.length) return possible[0];
        
        const counts = Array(6).fill().map(() => Array(26).fill(0));
        for (const word of possible) {
            for (let i = 0; i < 6; i++) {
                counts[i][word.charCodeAt(i) - 97]++;
            }
        }
        
        let maxScore = 0;
        let bestWord = candidates[0];
        
        for (const word of candidates) {
            let score = 0;
            for (let i = 0; i < 6; i++) {
                score += counts[i][word.charCodeAt(i) - 97];
            }
            if (score > maxScore) {
                maxScore = score;
                bestWord = word;
            }
        }
        
        return bestWord;
    };
    
    let possible = [...wordlist];
    
    for (let i = 0; i < 10 && possible.length; i++) {
        const guess = getMostCommonWord(possible, possible);
        const matchCount = master.guess(guess);
        
        if (matchCount === 6) return;
        
        possible = possible.filter(word => matches(word, guess) === matchCount);
    }
};

C# Solution

public class Solution {
    public void FindSecretWord(string[] wordlist, Master master) {
        var possible = new List(wordlist);
        
        for (int i = 0; i < 10 && possible.Count > 0; i++) {
            string guess = GetMostCommonWord(possible, possible);
            int matches = master.Guess(guess);
            
            if (matches == 6) return;
            
            possible = possible.Where(w => Matches(w, guess) == matches).ToList();
        }
    }
    
    private int Matches(string word1, string word2) {
        return word1.Zip(word2, (c1, c2) => c1 == c2 ? 1 : 0).Sum();
    }
    
    private string GetMostCommonWord(List candidates, List possible) {
        if (!candidates.Any()) {
            return possible[0];
        }
        
        var counts = new int[6][];
        for (int i = 0; i < 6; i++) {
            counts[i] = new int[26];
        }
        
        foreach (var word in possible) {
            for (int i = 0; i < 6; i++) {
                counts[i][word[i] - 'a']++;
            }
        }
        
        int maxScore = 0;
        string bestWord = candidates[0];
        
        foreach (var word in candidates) {
            int score = 0;
            for (int i = 0; i < 6; i++) {
                score += counts[i][word[i] - 'a'];
            }
            if (score > maxScore) {
                maxScore = score;
                bestWord = word;
            }
        }
        
        return bestWord;
    }
}

Implementation Notes:

  • Uses LINQ for efficient filtering and matching
  • Implements minimax strategy with frequency analysis
  • Time complexity: O(N²)
  • Space complexity: O(N)
