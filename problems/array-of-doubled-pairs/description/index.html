954. Array of Doubled Pairs

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer array of even length arr, return true if it is possible to reorder arr such that arr[2 * i + 1] = 2 * arr[2 * i] for every 0 <= i < len(arr) / 2, or false otherwise.

Example 1:

Input: arr = [3,1,3,6]
Output: false

Example 2:

Input: arr = [2,1,2,6]
Output: false

Example 3:

Input: arr = [4,-2,2,-4]
Output: true
Explanation: We can take two groups, [-2,-4] and [2,4] to form [-2,-4,2,4] or [2,4,-2,-4].

Constraints:

  • 2 <= arr.length <= 3 * 10^4
  • arr.length is even.
  • -10^5 <= arr[i] <= 10^5

Solution

Python Solution

class Solution:
    def canReorderDoubled(self, arr: List[int]) -> bool:
        # Count frequency of each number
        count = Counter(arr)
        
        # Sort by absolute value to handle negative numbers
        for x in sorted(arr, key=abs):
            if count[x] == 0:
                continue
            if count[2 * x] == 0:
                return False
            count[x] -= 1
            count[2 * x] -= 1
        
        return True

Time Complexity: O(n log n)

Where n is the length of arr. We need to sort the array and process each element once.

Space Complexity: O(n)

We need to store the frequency count of each number.

Java Solution

class Solution {
    public boolean canReorderDoubled(int[] arr) {
        Map count = new HashMap<>();
        for (int x : arr) {
            count.put(x, count.getOrDefault(x, 0) + 1);
        }
        
        // Sort by absolute value to handle negative numbers
        Integer[] sorted = Arrays.stream(arr).boxed().toArray(Integer[]::new);
        Arrays.sort(sorted, (a, b) -> Math.abs(a) - Math.abs(b));
        
        for (int x : sorted) {
            if (count.get(x) == 0) {
                continue;
            }
            if (count.getOrDefault(2 * x, 0) == 0) {
                return false;
            }
            count.put(x, count.get(x) - 1);
            count.put(2 * x, count.get(2 * x) - 1);
        }
        
        return true;
    }
}

C# Solution

public class Solution {
    public bool CanReorderDoubled(int[] arr) {
        var count = new Dictionary();
        foreach (int x in arr) {
            if (!count.ContainsKey(x)) {
                count[x] = 0;
            }
            count[x]++;
        }
        
        // Sort by absolute value to handle negative numbers
        Array.Sort(arr, (a, b) => Math.Abs(a).CompareTo(Math.Abs(b)));
        
        foreach (int x in arr) {
            if (count[x] == 0) {
                continue;
            }
            if (!count.ContainsKey(2 * x) || count[2 * x] == 0) {
                return false;
            }
            count[x]--;
            count[2 * x]--;
        }
        
        return true;
    }
}
