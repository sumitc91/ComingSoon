970. Powerful Integers

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given three integers x, y, and bound, return a list of all the powerful integers that have a value less than or equal to bound.

An integer is powerful if it can be represented as x^i + y^j for some integers i >= 0 and j >= 0.

You may return the answer in any order. In your answer, each value should occur at most once.

Example 1:

Input: x = 2, y = 3, bound = 10
Output: [2,3,4,5,7,9,10]
Explanation:
2 = 2^0 + 3^0
3 = 2^1 + 3^0
4 = 2^0 + 3^1
5 = 2^1 + 3^1
7 = 2^2 + 3^1
9 = 2^3 + 3^0
10 = 2^0 + 3^2

Example 2:

Input: x = 3, y = 5, bound = 15
Output: [2,4,6,8,10,14]

Constraints:

  • 1 <= x, y <= 100
  • 0 <= bound <= 10^6

Solution

Python Solution

class Solution:
    def powerfulIntegers(self, x: int, y: int, bound: int) -> List[int]:
        result = set()
        
        for i in range(20):
            for j in range(20):
                val = x**i + y**j
                if val <= bound:
                    result.add(val)
                else:
                    break
                    
        return list(result)

Time Complexity: O(log² bound)

We need to try all combinations of i and j up to log bound.

Space Complexity: O(log² bound)

We store all possible powerful integers in our set.

Java Solution

class Solution {
    public List powerfulIntegers(int x, int y, int bound) {
        Set result = new HashSet<>();
        
        for (int i = 0; i < 20; i++) {
            for (int j = 0; j < 20; j++) {
                int val = (int) (Math.pow(x, i) + Math.pow(y, j));
                if (val <= bound) {
                    result.add(val);
                } else {
                    break;
                }
            }
        }
        
        return new ArrayList<>(result);
    }
}
