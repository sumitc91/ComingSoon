696. Count Binary Substrings

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a binary string s, return the number of non-empty substrings that have the same number of 0's and 1's, and all the 0's and all the 1's in these substrings are grouped consecutively.

Substrings that occur multiple times are counted the number of times they occur.

Examples:

Example 1:

Input: s = "00110011"
Output: 6
Explanation: There are 6 substrings that have equal number of consecutive 1's and 0's: "0011", "01", "1100", "10", "0011", and "01".
Notice that some of these substrings repeat and are counted the number of times they occur.

Example 2:

Input: s = "10101"
Output: 4
Explanation: There are 4 substrings: "10", "01", "10", "01" that have equal number of consecutive 1's and 0's.

Constraints:

  • 1 ≤ s.length ≤ 105
  • s[i] is either '0' or '1'

Python Solution

class Solution:
    def countBinarySubstrings(self, s: str) -> int:
        prev_count = 0
        curr_count = 1
        result = 0
        
        for i in range(1, len(s)):
            if s[i] == s[i-1]:
                curr_count += 1
            else:
                result += min(prev_count, curr_count)
                prev_count = curr_count
                curr_count = 1
        
        # Add the last group
        result += min(prev_count, curr_count)
        return result

Implementation Notes:

  • Uses two counters to track consecutive characters
  • Time complexity: O(n)
  • Space complexity: O(1)

Java Solution

class Solution {
    public int countBinarySubstrings(String s) {
        int prevCount = 0;
        int currCount = 1;
        int result = 0;
        
        for (int i = 1; i < s.length(); i++) {
            if (s.charAt(i) == s.charAt(i-1)) {
                currCount++;
            } else {
                result += Math.min(prevCount, currCount);
                prevCount = currCount;
                currCount = 1;
            }
        }
        
        // Add the last group
        result += Math.min(prevCount, currCount);
        return result;
    }
}

C# Solution

public class Solution {
    public int CountBinarySubstrings(string s) {
        int prevCount = 0;
        int currCount = 1;
        int result = 0;
        
        for (int i = 1; i < s.Length; i++) {
            if (s[i] == s[i-1]) {
                currCount++;
            } else {
                result += Math.Min(prevCount, currCount);
                prevCount = currCount;
                currCount = 1;
            }
        }
        
        // Add the last group
        result += Math.Min(prevCount, currCount);
        return result;
    }
}

Implementation Notes:

  • Uses two variables to keep track of consecutive counts
  • Counts groups of consecutive characters and takes the minimum
  • Time complexity: O(n) where n is the length of the string
  • Space complexity: O(1) as we only use constant extra space
