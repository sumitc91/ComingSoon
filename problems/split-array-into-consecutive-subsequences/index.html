659. Split Array into Consecutive Subsequences

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an integer array nums that is sorted in non-decreasing order.

Determine if it is possible to split nums into one or more subsequences such that both of these conditions are true:

  • Each subsequence is a consecutive increasing sequence (i.e. each integer is exactly one more than the previous integer).
  • All subsequences have a length of 3 or more.

Return true if you can split nums according to the above conditions, or false otherwise.

Examples:

Example 1:

Input: nums = [1,2,3,3,4,5]
Output: true
Explanation: nums can be split into the following subsequences:
[1,2,3,4,5] and [3]
[1,2,3] and [3,4,5]

Example 2:

Input: nums = [1,2,3,3,4,4,5,5]
Output: true
Explanation: nums can be split into the following subsequences:
[1,2,3,4,5] and [3,4,5]

Example 3:

Input: nums = [1,2,3,4,4,5]
Output: false
Explanation: It is impossible to split nums into consecutive increasing subsequences of length 3 or more.

Constraints:

  • 1 ≤ nums.length ≤ 104
  • -1000 ≤ nums[i] ≤ 1000
  • nums is sorted in non-decreasing order

Python Solution

class Solution:
    def isPossible(self, nums: List[int]) -> bool:
        # Count frequency of each number
        count = Counter(nums)
        # Track end counts of sequences
        end = Counter()
        
        # Try to add each number to an existing sequence
        # or start a new sequence
        for num in nums:
            if count[num] == 0:
                continue
                
            # If can append to existing sequence
            if end[num - 1] > 0:
                end[num - 1] -= 1
                end[num] += 1
            else:
                # Try to start new sequence of length 3
                if count[num + 1] == 0 or count[num + 2] == 0:
                    return False
                count[num + 1] -= 1
                count[num + 2] -= 1
                end[num + 2] += 1
            
            count[num] -= 1
        
        return True

Alternative Solution (Using Priority Queue):

from heapq import heappush, heappop

class Solution:
    def isPossible(self, nums: List[int]) -> bool:
        sequences = []  # min heap of (end_val, length) tuples
        
        for num in nums:
            # Remove sequences that can't be extended
            while sequences and sequences[0][0] + 1 < num:
                if sequences[0][1] < 3:
                    return False
                heappop(sequences)
            
            if not sequences or sequences[0][0] == num:
                # Start new sequence
                heappush(sequences, (num, 1))
            else:  # sequences[0][0] + 1 == num
                # Extend existing sequence
                _, length = heappop(sequences)
                heappush(sequences, (num, length + 1))
        
        # Check remaining sequences
        return all(length >= 3 for _, length in sequences)

Implementation Notes:

  • First solution uses two counters with O(n) time and space
  • Second solution uses priority queue with O(nlogn) time
  • Counter solution is more efficient for this problem

Java Solution

class Solution {
    public boolean isPossible(int[] nums) {
        Map count = new HashMap<>();
        Map end = new HashMap<>();
        
        // Count frequency of each number
        for (int num : nums) {
            count.put(num, count.getOrDefault(num, 0) + 1);
        }
        
        // Try to add each number to an existing sequence
        // or start a new sequence
        for (int num : nums) {
            if (count.get(num) == 0) {
                continue;
            }
            
            // If can append to existing sequence
            if (end.getOrDefault(num - 1, 0) > 0) {
                end.put(num - 1, end.get(num - 1) - 1);
                end.put(num, end.getOrDefault(num, 0) + 1);
            } else {
                // Try to start new sequence of length 3
                if (count.getOrDefault(num + 1, 0) == 0 || 
                    count.getOrDefault(num + 2, 0) == 0) {
                    return false;
                }
                count.put(num + 1, count.get(num + 1) - 1);
                count.put(num + 2, count.get(num + 2) - 1);
                end.put(num + 2, end.getOrDefault(num + 2, 0) + 1);
            }
            
            count.put(num, count.get(num) - 1);
        }
        
        return true;
    }
}

    const end = new Map();
    
    // Count frequency of each number
    for (const num of nums) {
        count.set(num, (count.get(num) || 0) + 1);
    }
    
    // Try to add each number to an existing sequence
    // or start a new sequence
    for (const num of nums) {
        if (count.get(num) === 0) {
            continue;
        }
        
        // If can append to existing sequence
        if ((end.get(num - 1) || 0) > 0) {
            end.set(num - 1, end.get(num - 1) - 1);
            end.set(num, (end.get(num) || 0) + 1);

            // Try to start new sequence of length 3

            end.set(num + 2, (end.get(num + 2) || 0) + 1);
        }
        
        count.set(num, count.get(num) - 1);
    }
    
    return true;

C# Solution

public class Solution {
    public bool IsPossible(int[] nums) {
        Dictionary count = new Dictionary();
        Dictionary end = new Dictionary();
        
        // Count frequency of each number
        foreach (int num in nums) {
            if (!count.ContainsKey(num)) {
                count[num] = 0;
            }
            count[num]++;
        }
        
        // Try to add each number to an existing sequence
        // or start a new sequence
        foreach (int num in nums) {
            if (count[num] == 0) {
                continue;
            }
            
            // If can append to existing sequence
            if (end.ContainsKey(num - 1) && end[num - 1] > 0) {
                end[num - 1]--;
                if (!end.ContainsKey(num)) {
                    end[num] = 0;
                }
                end[num]++;
            } else {
                // Try to start new sequence of length 3
                if (!count.ContainsKey(num + 1) || count[num + 1] == 0 ||
                    !count.ContainsKey(num + 2) || count[num + 2] == 0) {
                    return false;
                }
                count[num + 1]--;
                count[num + 2]--;
                if (!end.ContainsKey(num + 2)) {
                    end[num + 2] = 0;
                }
                end[num + 2]++;
            }
            
            count[num]--;
        }
        
        return true;
    }
}

Implementation Notes:

  • Uses two dictionaries to track number frequencies and sequence ends
  • Handles dictionary key existence checks explicitly
  • Time complexity is O(n), space complexity is O(n)
